# [34956. Longest Common Path](https://www.acmicpc.net/problem/34956)

## 문제
$1$부터 $N$까지 번호가 붙은 $N$개 정점으로 이루어진 두 개의 트리 $T_1$, $T_2$가 주어진다. 서로 다른 양의 정수 $s$, $e$에 대해 각 트리에서 $s$번 정점에서 $e$번 정점으로 가는 최단 경로를 생각하자. 두 트리에서 최단 경로를 이루는 정점들의 번호 순서가 완전히 동일할 때, 그 경로를 두 트리의 공통 경로(Common Path)라고 정의한다. 이때 공통 경로의 길이는 $s$번 정점과 $e$번 정점을 잇는 최단 경로에 포함되는 간선의 수이다.


두 트리에서 찾을 수 있는 공통 경로 중 가장 길이가 긴 것을 하나 찾는 프로그램을 작성하시오.

## 예제 입력 1
```
8
1 2
1 3
1 4
2 5
2 6
2 7
5 8
1 2
1 3
2 4
2 5
3 6
4 7
5 8
```

## 예제 출력 1
```
8 3
```
