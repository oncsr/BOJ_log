# [11023. 더하기 3](https://www.acmicpc.net/problem/11023)

## 문제
수 N개가 주어졌을 때, N개의 합을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
1 2 3 4 5
```

## 예제 출력 1
```
15
```

## 예제 입력 2
```
5 4 5 4 2 3 1 2
```

## 예제 출력 2
```
26
```
