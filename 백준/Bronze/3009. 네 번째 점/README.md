# [3009. 네 번째 점](https://www.acmicpc.net/problem/3009)

## 문제
세 점이 주어졌을 때, 축에 평행한 직사각형을 만들기 위해서 필요한 네 번째 점을 찾는 프로그램을 작성하시오.

## 예제 입력 1
```
5 5
5 7
7 5
```

## 예제 출력 1
```
7 7
```

## 예제 입력 2
```
30 20
10 10
10 20
```

## 예제 출력 2
```
30 10
```
