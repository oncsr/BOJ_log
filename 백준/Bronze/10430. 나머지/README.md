# [10430. 나머지](https://www.acmicpc.net/problem/10430)

## 문제
(A+B)%C는 ((A%C) + (B%C))%C 와 같을까?


(A×B)%C는 ((A%C) × (B%C))%C 와 같을까?


세 수 A, B, C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
5 8 4
```

## 예제 출력 1
```
1
1
0
0
```
