# [34750. 추석은 언제나 좋아](https://www.acmicpc.net/problem/34750)

## 문제
군대를 다녀와 철이 든 재민이는 추석마다 할아버지께 $10\,000$원 단위로 용돈을 받는데, 이 중 일부는 감사한 마음으로 부모님께 드리고 나머지는 자신이 갖는다.


부모님께 드리는 금액은 할아버지께 받은 용돈의 액수에 따라 다음과 같은 규칙으로 정해진다.




받은 금액이 $1\,000\,000$원 이상일 경우: 금액의 $20\%$


받은 금액이 $500\,000$원 이상 $1\,000\,000$원 미만일 경우: 금액의 $15\%$


받은 금액이 $100\,000$원 이상 $500\,000$원 미만일 경우: 금액의 $10\%$


받은 금액이 $100\,000$원 미만일 경우: 금액의 $5\%$




할아버지께 받은 용돈 액수 $N$이 주어졌을 때, 재민이가 부모님께 드리는 금액과 재민이가 최종적으로 갖게 되는 금액을 계산하는 프로그램을 작성해 보자.

## 예제 입력 1
```
200000
```

## 예제 출력 1
```
20000 180000
```

## 예제 입력 2
```
800000
```

## 예제 출력 2
```
120000 680000
```
