# [1264. 모음의 개수](https://www.acmicpc.net/problem/1264)

## 문제
영문 문장을 입력받아 모음의 개수를 세는 프로그램을 작성하시오. 모음은 'a', 'e', 'i', 'o', 'u'이며 대문자 또는 소문자이다.

## 예제 입력 1
```
How are you today?
Quite well, thank you, how about yourself?
I live at number twenty four.
#
```

## 예제 출력 1
```
7
14
9
```
