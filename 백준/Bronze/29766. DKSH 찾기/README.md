# [29766. DKSH 찾기](https://www.acmicpc.net/problem/29766)

## 문제
학교의 로고인 DKSH는 Dankook University Software High School의 약자이다.


D
, 
K
, 
S
, 
H
로만 이루어진 문자열이 주어진다. 이 문자열에서 
DKSH
가 몇 번 나타나는지 구해보자.

## 예제 입력 1
```
DKKSSH
```

## 예제 출력 1
```
0
```

## 예제 입력 2
```
HDKSHDDKS
```

## 예제 출력 2
```
1
```

## 예제 입력 3
```
SDKSHSSDKSHS
```

## 예제 출력 3
```
2
```
