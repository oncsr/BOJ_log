# [24350. НУЛИ](https://www.acmicpc.net/problem/24350)

## 문제
Дадени са две цели числа $N$ и $K$. Напишете програма zeros, която намира броя на нулите в десетичния запис на числото $C_N^K$.


$C_N^K$ – брой на различните начини да се изберат $K$ предмета от $N$ различни, без да се отчита наредбата им.


$$C_N^K = \frac{N!}{K!(N-K)!}$$

## 예제 입력 1
```
50 40
```

## 예제 출력 1
```
2
```
