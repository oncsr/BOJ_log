# [13235. 팰린드롬](https://www.acmicpc.net/problem/13235)

## 문제
팰린드롬은 앞에서부터 읽을 때와 뒤에서부터 읽을 때가 똑같은 단어를 의미한다. 예를 들어, eve, eevee는 팰린드롬이고, eeve는 팰린드롬이 아니다. 단어가 주어졌을 때, 팰린드롬인지 아닌지 판단해보자.

## 예제 입력 1
```
a
```

## 예제 출력 1
```
true
```

## 예제 입력 2
```
uu
```

## 예제 출력 2
```
true
```

## 예제 입력 3
```
owo
```

## 예제 출력 3
```
true
```

## 예제 입력 4
```
bbbbbbbbbbb
```

## 예제 출력 4
```
true
```

## 예제 입력 5
```
zzzzzzzzo
```

## 예제 출력 5
```
false
```

## 예제 입력 6
```
slakdjfims
```

## 예제 출력 6
```
false
```
