# [14264. 정육각형과 삼각형](https://www.acmicpc.net/problem/14264)

## 문제
한 변의 길이가 L인 정육각형을 가지고 있다. 오늘은 이 정육각형에 세 개의 겹치지 않는 대각선을 그리려고 한다.


대각선은 정육각형을 4개의 삼각형으로 나눈다. 4개의 삼각형 중에 면적이 제일 작은 삼각형의 면적을 S라고 한다.


가능한 S중에서 최댓값을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
5
```

## 예제 출력 1
```
10.825317547305485
```

## 예제 입력 2
```
10
```

## 예제 출력 2
```
43.30127018922194
```

## 예제 입력 3
```
100000
```

## 예제 출력 3
```
4.330127018922194E9
```
