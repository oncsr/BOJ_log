# [3226. 전화 요금](https://www.acmicpc.net/problem/3226)

## 문제
7:00부터 19:00까지 전화 요금은 1분에 10원이다.


19:00부터 7:00까지 전화 요금은 1분에 5원이다.


상근이가 전화를 건 시간과 통화 시간이 모두 주어졌을 때, 전화 요금을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
2
11:02 11
15:30 01
```

## 예제 출력 1
```
120
```

## 예제 입력 2
```
3
20:05 12
06:45 30
13:08 15
```

## 예제 출력 2
```
435
```

## 예제 입력 3
```
5
00:00 05
06:47 35
11:30 18
18:33 60
23:59 22
```

## 예제 출력 3
```
1035
```
