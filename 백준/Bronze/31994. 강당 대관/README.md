# [31994. 강당 대관](https://www.acmicpc.net/problem/31994)

## 문제
한국정보기술진흥원에서는 다양한 세미나가 열린다.


전문가를 위한 알고리즘, 데이터분석, 인공지능, 사이버보안, 네트워크 세미나뿐만 아니라 예비 창업가를 위한 창업 세미나, 그리고 청소년들을 위한 입시 세미나가 열린다
