# [34032. /gg](https://www.acmicpc.net/problem/34032)

## 문제
haru_101
이 플레이하는 게임에 
/gg
 명령어가 있다. 이 명령어는 항복 투표를 진행하기 위한 명령어로, 해당 팀의 각 팀원은 
/gg
를 입력함으로써 항복 의사를 밝힐 수 있다. 팀원 중 절반 이상이 
/gg
 명령어를 입력하면 팀은 항복하게 된다.


각 팀원의 명령어 입력 여부가 주어졌을 때, 해당 팀이 항복하게 되는지를 알아보자.

## 예제 입력 1
```
5
OOOOX
```

## 예제 출력 1
```
Yes
```

## 예제 입력 2
```
7
XOXOXOX
```

## 예제 출력 2
```
No
```
