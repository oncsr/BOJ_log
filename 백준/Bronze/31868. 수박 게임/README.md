# [31868. 수박 게임](https://www.acmicpc.net/problem/31868)

## 문제
$1$단계 과일은 체리, $N$단계 과일은 수박이다.


$i$단계 과일 $2$개를 소모하여 $i+1$단계 과일 $1$개를 만들 수 있다. $(1 \leq i \leq N-1)$


$K$개의 체리로 최대 몇 개의 수박을 만들 수 있는지 구해보자!

## 예제 입력 1
```
3 10
```

## 예제 출력 1
```
2
```
