# [27324. ゾロ目 (Same Numbers)](https://www.acmicpc.net/problem/27324)

## 문제
十進法で 
2
 桁の整数 
N
 が与えられる．


N
 の十の位の数字と一の位の数字が同じである場合は 
1
 を，そうでない場合は 
0
 を出力せよ．

## 예제 입력 1
```
22
```

## 예제 출력 1
```
1
```

## 예제 입력 2
```
10
```

## 예제 출력 2
```
0
```
