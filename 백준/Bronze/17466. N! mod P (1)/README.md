# [17466. N! mod P (1)](https://www.acmicpc.net/problem/17466)

## 문제
양의 정수 N과, N보다 큰 소수 P가 주어질 때, N!을 P로 나눈 나머지를 구하여라.

## 예제 입력 1
```
4 7
```

## 예제 출력 1
```
3
```

## 예제 입력 2
```
99999988 99999989
```

## 예제 출력 2
```
99999988
```
