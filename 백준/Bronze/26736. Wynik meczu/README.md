# [26736. Wynik meczu](https://www.acmicpc.net/problem/26736)

## 문제
Drużyny Algogrodu i Bajtocji grają w wielkim finale Mistrzostw Świata w Piłce Ręcznej. Bajtosia, jako analityk sportowy, skrzętnie notuje wszystkie gole. Za każdym razem, kiedy padnie gol dla jednej z drużyn, Bajtosia zapisuje na kartce literkę 
A
 albo 
B
, oznaczające odpowiednio gol dla Algogrodu oraz gol dla Bajtocji.


Jeden z zawodników przypadkowo trafił piłką w ekran wyświetlający wynik meczu. Czy na podstawie notatek Bajtosi jesteś w stanie powiedzieć jaki jest aktualny wynik?

## 예제 입력 1
```
AAABBBA
```

## 예제 출력 1
```
4 : 3
```

## 예제 입력 2
```
BABABBA
```

## 예제 출력 2
```
3 : 4
```

## 예제 입력 3
```
B
```

## 예제 출력 3
```
0 : 1
```
