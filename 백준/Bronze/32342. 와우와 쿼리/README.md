# [32342. 와우와 쿼리](https://www.acmicpc.net/problem/32342)

## 문제
홍익대학교에는 마스코트 와우가 있다. 와우를 본 사람들은 너무 귀여워서 소리를 지르곤 한다. 당신은 소리를 연구하다 보니 와우의 귀여움의 정도를 분석할 수 있게 되었다.


사람들이 내는 소리를 문자열 $S$라고 했을 때, $S$에서 
WOW
라는 부분 문자열의 개수가 곧 현재 와우의 귀여움임을 알게 되었다. 총 $Q$개의 소리가 주어졌을 때, 각 소리별 와우의 귀여움을 알아내보자!

## 예제 입력 1
```
4
WOW
WWO
WOWOWWOW
OWOOWOOW
```

## 예제 출력 1
```
1
0
3
0
```
