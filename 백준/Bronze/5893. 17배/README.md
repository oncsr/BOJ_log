# [5893. 17배](https://www.acmicpc.net/problem/5893)

## 문제
상근이는 이진수 곱셈에 어려움을 겪는 여자친구를 위한 프로그램을 만들려고 한다.


상근이의 여자친구는 항상 이진수에 17을 곱한다. 즉, 이진수 N이 입력으로 들어오면 17을 곱한 다음 이진수로 출력하는 프로그램을 작성하시오.

## 예제 입력 1
```
10110111
```

## 예제 출력 1
```
110000100111
```
