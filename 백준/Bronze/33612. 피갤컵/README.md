# [33612. 피갤컵](https://www.acmicpc.net/problem/33612)

## 문제
제1회 피갤컵이 2024년 8월에 열렸다.  


제2회 피갤컵은 2025년 3월에 열린다.  


이를 통해 피갤컵은 7개월 주기로 열린다는 사실을 알 수 있다.  


그렇다면 $N$번째 피갤컵은 언제 열릴까?

## 예제 입력 1
```
1
```

## 예제 출력 1
```
2024 8
```

## 예제 입력 2
```
3
```

## 예제 출력 2
```
2025 10
```
