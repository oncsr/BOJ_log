# [30045. ZOAC 6](https://www.acmicpc.net/problem/30045)

## 문제
2023년 9월, 여섯 번째로 개최된 ZOAC의 오프닝을 또 맡은 성우는 영과일의 마스코트인 영일이를 이용해 대회를 홍보하기로 했다.


성우는 홍보 글이 주어질 때 각 문장에 
01
 또는 
OI
가 포함되어 있다면 문장 끝에 한 개의 영일이 이모티콘을 넣기로 했다. 이때, 홍보 글에 영일이 이모티콘을 총 몇 번 넣어야 하는지 구하여라.

## 예제 입력 1
```
4
zoacisgoodcontest
2018to2023
hello
JOIN
```

## 예제 출력 1
```
2
```

## 예제 입력 2
```
4
SPOILER
HARD
OI01OI
OioioI
```

## 예제 출력 2
```
2
```
