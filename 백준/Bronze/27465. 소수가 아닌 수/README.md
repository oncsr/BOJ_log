# [27465. 소수가 아닌 수](https://www.acmicpc.net/problem/27465)

## 문제
이 대회의 운영진 중 한 명인 KSA 학생은 $17$시와 $19$시를 구별할 수 없다. 이는 당연하게도 $17$과 $19$가 모두 소수이기 때문일 것이다. 시간을 제대로 구별해서 KSA의 명예를 지키기 위해 정수 $N$을 입력받아서 소수가 아닌 $N$ 이상의 정수를 아무거나 구해주자.

## 예제 입력 1
```
5
```

## 예제 출력 1
```
10
```

## 예제 입력 2
```
6
```

## 예제 출력 2
```
12
```
