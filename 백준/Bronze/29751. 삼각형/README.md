# [29751. 삼각형](https://www.acmicpc.net/problem/29751)

## 문제

## 예제 입력 1
```
1 1
```

## 예제 출력 1
```
0.5
```

## 예제 입력 2
```
2 3
```

## 예제 출력 2
```
3.0
```
