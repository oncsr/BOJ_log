# [10179. 쿠폰](https://www.acmicpc.net/problem/10179)

## 문제
당신은 어떤 물건이라도 20% 할인해주는 쿠폰을 가지고 있다.


원래 가격이 주어질 때, 쿠폰을 사용하면 얼마가 되는지 알려주는 프로그램을 작성하시오.

## 예제 입력 1
```
3
100.00
59.99
20.00
```

## 예제 출력 1
```
$80.00
$47.99
$16.00
```
