# [11721. 열 개씩 끊어 출력하기](https://www.acmicpc.net/problem/11721)

## 문제
알파벳 소문자와 대문자로만 이루어진 길이가 N인 단어가 주어진다.


한 줄에 10글자씩 끊어서 출력하는 프로그램을 작성하시오.

## 예제 입력 1
```
BaekjoonOnlineJudge
```

## 예제 출력 1
```
BaekjoonOn
lineJudge
```

## 예제 입력 2
```
OneTwoThreeFourFiveSixSevenEightNineTen
```

## 예제 출력 2
```
OneTwoThre
eFourFiveS
ixSevenEig
htNineTen
```
