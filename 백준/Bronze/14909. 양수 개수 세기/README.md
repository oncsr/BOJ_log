# [14909. 양수 개수 세기](https://www.acmicpc.net/problem/14909)

## 문제
주어진 N개의 정수 중에서 양의 정수의 개수를 출력하는 프로그램을 작성하시오.

## 예제 입력 1
```
3 9 11 32 8 2 6
```

## 예제 출력 1
```
7
```

## 예제 입력 2
```
-2 0 21 3 8 17 32 -8 7 0
```

## 예제 출력 2
```
6
```

## 예제 입력 3
```
0
```

## 예제 출력 3
```
0
```
