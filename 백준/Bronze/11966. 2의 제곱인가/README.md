# [11966. 2의 제곱인가?](https://www.acmicpc.net/problem/11966)

## 문제

```
1
```

## 예제 출력 1

```
1
```

## 예제 입력 2
```
2
```

## 예제 출력 2
```
1
```

## 예제 입력 3
```
3
```

## 예제 출력 3
```
0
```

## 예제 입력 4
```
4
```

## 예제 출력 4
```
1
```
