# [31669. 특별한 학교 탈출](https://www.acmicpc.net/problem/31669)

## 문제
NLCS Jeju의 학생들은 수업에 질려 학교에서 탈출하고 싶어한다. 수업 시간에 학교에서 탈출하는 것은 어려운 일이다. 


NLCS Jeju의 수업 시간은 $1$교시부터 $M$교시이다. 또한 NLCS Jeju에 있는 $N$명의 선생님들은 각각 $1$교시부터 $M$교시 사이에 정해진 스케줄에 따라 해당 수업 시간 동안 순찰한다. 학교에서 탈출하다 선생님에게 들키면 안 되기 때문에 학생들은 아무도 순찰하지 않을 때 학교에서 탈출해야 한다. 학교 탈출을 시도하려는 시각에 순찰하는 선생님이 있다면 선생님에게 무조건 들킨다.


각 선생님의 순찰 스케줄이 주어질 때 학생들이 학교에서 탈출할 수 있는지 판단하고, 가능하다면 학교에서 탈출할 수 있는 가장 빠른 시각을 구하자.

## 예제 입력 1
```
5 5
OXXOO
XXXOX
OOXXX
OXXOO
XOXXO
```

## 예제 출력 1
```
3
```

```
5 5
OXXXX
XOXXX
XXOXX
XXXOX
XXXXO
```

```
ESCAPE FAILED
```

```
7 5
OOXXO
OXXOO
XOXXX
XOOOX

OXOOO
XOOXO
```

```
ESCAPE FAILED
```
