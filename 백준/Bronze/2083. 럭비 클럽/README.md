# [2083. 럭비 클럽](https://www.acmicpc.net/problem/2083)

## 문제
올 골드 럭비 클럽의 회원들은 성인부 또는 청소년부로 분류된다.


나이가 17세보다 많거나, 몸무게가 80kg 이상이면 성인부이다. 그 밖에는 모두 청소년부이다. 클럽 회원들을 올바르게 분류하라.

## 예제 입력 1
```
Joe 16 34
Bill 18 65
Billy 17 65
Sam 17 85
# 0 0
```

## 예제 출력 1
```
Joe Junior
Bill Senior
Billy Junior
Sam Senior
```
