# [11932. 트리와 K번째 수](https://www.acmicpc.net/problem/11932)

## 문제
1번부터 N번까지 번호가 붙여진 N개의 정점과 N-1개의 간선으로 구성된 트리가 있다. 트리의 각 정점에는 가중치가 있다. 이때 M개의 질문에 대해 다음의 연산을 수행해야 한다.


연산 X Y K : 정점 X와 Y를 잇는 경로 상에서 K번째로 작은 가중치를 출력한다.

## 예제 입력 1
```
3 2
1 2 3
1 2
3 1
2 3 1
1 3 2
```

## 예제 출력 1
```
1
3
```
