# [13557. 수열과 쿼리 10](https://www.acmicpc.net/problem/13557)

## 문제
길이가 N인 수열 A
1
, A
2
, ..., A
N
이 주어진다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오.




x1 y1 x2 y2
: x1 ≤ i ≤ y1, x2 ≤ j ≤ y2, i ≤ j인 모든 (i, j)에 대해서 A
i
 + ... + A
j
의 최댓값을 출력한다. (1 ≤ x1 ≤ x2 ≤ N, 1 ≤ y1 ≤ y2 ≤ N, x1 ≤ y1, x2 ≤ y2)

## 예제 입력 1
```
6
3 -2 1 -4 5 2
2
1 1 2 3
1 3 2 5
```

## 예제 출력 1
```
2
3
```

## 예제 입력 2
```
1
1
1
1 1 1 1
```

## 예제 출력 2
```
1
```
