# [16901. XOR MST](https://www.acmicpc.net/problem/16901)

## 문제
N개의 정점으로 이루어진 방향없는 그래프가 있다. i번째 정점에는 정수 A
i
가 적혀있다. i번 정점과 j번 정점을 연결하는 간선의 가중치는 A
i
 xor A
j
 이다.


이 그래프의 최소 스패닝 트리(MST)의 비용을 구하시오.

## 예제 입력 1
```
5
1 2 3 4 5
```

## 예제 출력 1
```
8
```

## 예제 입력 2
```
4
1 2 3 4
```

## 예제 출력 2
```
8
```
