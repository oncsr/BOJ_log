# [17101. Dynamic Centroid](https://www.acmicpc.net/problem/17101)

## 문제
크기 N의 트리에서, 센트로이드란 정점을 기준으로 나누어지는 서브트리의 크기가 모두 N/2 이하인 정점을 뜻한다. 어떠한 트리라도 센트로이드가 존재함이 알려져있다.


1번 정점이 트리의 루트이며, i번 정점의 부모는 p
i
이고, p
i
<i이다.


1부터 N까지의 k에 대해, 1번 정점부터 k번 정점까지만 사용한 트리의 센트로이드를 구하여라.

## 예제 입력 1
```
5
1 2 3 4
```

## 예제 출력 1
```
1 1 2 2 3
```

## 예제 입력 2
```
5
1 2 1 4
```

## 예제 출력 2
```
1 1 2 1 1
```
