### 문제 설명

<p><em>주의! 이 문제는 NP-hard로 밝혀졌습니다. 하지만 NP-hard 문제를 출제하면 안 된다는 규정이 없었기 때문에 그냥 두기로 했습니다.</em></p>

<p>정점 <em>n</em>개, 간선 <em>m</em>개로 이루어진 양방향 그래프가 있다. 정점은 1부터 <em>n</em>까지, 간선은 1부터 <em>m</em>까지 번호가 메겨져 있으며 <em>i</em>번 간선의 가중치는 <em>w<sub>i</sub></em>이다. (1 ≤ <em>i</em> ≤ <em>m</em>) 이때 자연수 <em>k</em>에 대해서, 1번 정점에서 시작해서 <em>n</em>번 정점에서 끝나는 간선 <em>k</em>개로 이루어진 최단 단순 경로의 길이를 구하여라.</p>

<p>단순 경로란 같은 정점을 두 번 지나지 않는 경로를 의미하며, 경로의 길이는 해당 경로를 구성하는 간선들의 가중치 합을 의미한다.</p>

### 입력 

 <p>첫째 줄에 세 정수 <em>n</em>, <em>m</em>, <em>k</em>가 공백으로 구분되어 주어진다.</p>

<p>다음 <em>m</em>개의 줄 중 <em>i</em>번째 줄에는 세 정수 <em>x<sub>i</sub></em>, <em>y<sub>i</sub></em>, <em>w<sub>i</sub></em>가 공백으로 구분되어 주어진다. 이는 <em>i</em>번 간선이 <em>x<sub>i</sub></em>번 정점과 <em>y<sub>i</sub></em>번 정점을 잇는 가중치 <em>w<sub>i</sub></em>의 간선이라는 것을 의미한다.</p>

<p>루프나 다중간선은 주어지지 않는다.</p>

### 출력 

 <p>1번 정점에서 시작해서 <em>n</em>번 정점에서 끝나는 간선 <em>k</em>개로 이루어진 최단 단순 경로의 길이를 출력하라. 단, 그러한 경로가 없다면 <code>-1</code>을 출력하라.</p>
