# [12858. Range GCD](https://www.acmicpc.net/problem/12858)

## 문제
N개의 자연수로 이루어진 수열이 있다. 이 수열의 연속한 일부분에 일정한 값을 더하는 연산과 연속한 일부분의 최대공약수를 구하는 연산을 수행하자.

## 예제 입력 1
```
4
6 3 38 49
5
0 1 3
9 2 2
0 1 2
6 3 3
0 3 4
```

## 예제 출력 1
```
1
6
1
```
