### 문제 설명

<p>게임 마스터 <code><span style="color:#e74c3c;">amsminn</span></code>은 유망주 <code><span style="color:#e74c3c;">bnb2011</span></code>과 게임을 하려고 한다. 게임은 $1$ 이상 $N$ 이하의 정수 중 하나가 적혀있는 카드 $M$개가 펼쳐져 있는 책상에서 진행되며, 게임의 규칙은 다음과 같다.</p>

<ol>
	<li>게임은 교대로 진행되며, <code><span style="color:#e74c3c;">amsminn</span></code>이 먼저 게임을 시작한다.</li>
	<li>각 사람은 자기 차례가 왔을 때, 책상에 남아 있는 카드 중 하나를 가져간다. 이때, 가져간 카드는 이전까지 서로가 가져간 카드 전부와 서로소여야 한다. 그리고 가져간 카드는 책상에서 사라지게 된다.</li>
	<li>자신의 차례가 되었을 때 더 이상 카드가 존재하지 않거나, 규칙에 위배되지 않는 카드를 가져갈 수 없는 사람이 지게 된다.</li>
</ol>

<p>두 수의 최대공약수가 $1$인 경우 두 수를 서로소라고 한다. 서로가 최선을 다해 게임을 진행한다고 했을 때 누가 이기는지 구해보자.</p>

### 입력 

 <p>첫째 줄에 테스트 케이스의 수 $T$가 주어진다. $(1 \le T \le 10)$</p>

<p>각 테스트 케이스는 두 줄에 걸쳐 주어진다. 첫째 줄에 카드에 적힐 수 있는 수의 최댓값인 $N$과 카드의 개수인 $M$이 공백으로 구분되어 주어진다. $(1 \le N \le 150$; $1 \le M \le 100\,000)$</p>

<p>그리고 둘째 줄에 각 카드에 적혀 있는 정수가 공백으로 구분되어 주어진다. 카드에 적힌 수는 $1$ 이상 $N$ 이하의 정수 중 하나이다.</p>

<p>모든 테스트 케이스에 대한 $N$의 합은 $150$ 이하이며, 입력으로 주어지는 모든 수는 정수이다.</p>

### 출력 

 <p>각 테스트 케이스에 대해 <code><span style="color:#e74c3c;">amsminn</span></code>이 이긴다면 <code><span style="color:#e74c3c;">amsminn</span></code>를, <code><span style="color:#e74c3c;">bnb2011</span></code>이 이긴다면 <code><span style="color:#e74c3c;">bnb2011</span></code>를 각 줄에 출력한다.</p>
