# [24547. mod와 쿼리](https://www.acmicpc.net/problem/24547)

## 문제
길이가 $N$인 양의 정수들의 수열 $A_1, A_2, ... A_N$이 주어진다. 이때, 다음과 같은 쿼리를 수행하는 프로그램을 작성하시오. 




$1$ $X$ : 주어지는 양의 정수 $X$에 대해 $ \sum_{i=1}^N {({A_i}\bmod{X})} $를 출력한다.


$2$ $X$ : 주어지는 양의 정수 $X$에 대해 $ \sum_{i=1}^N {({X}\bmod{A_i})} $를 출력한다.


$3$ $i$ $X$ : $A_i$의 값을 양의 정수 $X$로 바꾼다. ($1\le i \le N$) 




${P}\bmod{Q}$는 $P$를 $Q$로 나눈 나머지를 뜻한다.

## 예제 입력 1
```
5
1 2 3 4 5
4
1 2
2 3
3 1 7
2 5
```

## 예제 출력 1
```
3
7
9
```
