# [3988. 수 고르기](https://www.acmicpc.net/problem/3988)

## 문제
수열이 주어졌을 때, M을 수열의 모든 두 원소의 차이 중 가장 큰 값이라고 한다. m은 그 차이 중 가장 작은 값이라고 한다.


크기가 N인 수열 V가 주어진다. 여기서 K개 수를 적절히 제거해서 M+m을 가능한 작게 만드는 프로그램을 작성하시오.

## 예제 입력 1
```
5 2
-3 -2 3 8 6
```

## 예제 출력 1
```
7
```

## 예제 입력 2
```
6 2
-5 8 10 1 13 -1
```

## 예제 출력 2
```
13
```

## 예제 입력 3
```
6 3
10 2 8 17 2 17
```

## 예제 출력 3
```
6
```
