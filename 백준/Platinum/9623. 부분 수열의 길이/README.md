# [9623. 부분 수열의 길이](https://www.acmicpc.net/problem/9623)

## 문제
길이가 N인 정수 수열과 정수 X가 주어진다. 이때, 합이 X보다 크거나 같은 연속 부분 수열 중에서 길이가 가장 짧은 것을 찾는 프로그램을 작성하시오.

## 예제 입력 1
```
3
5 4
1 2 1 2 1
6 -2
-5 -6 -7 -8 -9 -10
5 3
-1 1 1 1 -1
```

## 예제 출력 1
```
3
-1
3
```
