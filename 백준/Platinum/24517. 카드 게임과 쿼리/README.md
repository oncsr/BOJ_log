# [24517. 카드 게임과 쿼리](https://www.acmicpc.net/problem/24517)

## 문제
swoon은 raararaara와 게임을 하고 있다. 게임의 내용은 다음과 같다. 정수 $A$와 $B$가 주어지며, $A$를 $B$ 이상으로 만드는 사람이 승리한다.


양 플레이어 모두가 사용 가능한 $K$개의 카드가 놓여있으며, 각 카드에는 $1$부터 $K$까지의 정수가 하나씩 적혀 있다. 플레이어는 자신의 차례에 카드를 한 장 선택해 해당 카드에 적혀있는 수만큼 $A$에 더해야 한다. 선택한 카드는 사라진다. 사용할 수 있는 카드가 없다면, 다시 $K$개의 카드를 놓는다. 


게임은 항상 swoon부터 시작하며, 모든 플레이어는 최적의 방법으로 게임을 한다.


swoon은 이 게임을 무조건 이기고 싶기 때문에 $A$, $B$, $K$가 주어졌을 때, 승자가 누구인지 알고 싶다. swoon을 위해 승자를 알려주는 프로그램을 작성해보자.

## 예제 입력 1
```
2
2 4 7
4 10 5
```

## 예제 출력 1
```
swoon
raararaara
```
