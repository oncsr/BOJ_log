# [30629. 재미없는 문제](https://www.acmicpc.net/problem/30629)

## 문제
이 문제는 재미없다. 보통 사람들은 재미없는 문제를 풀지 않겠지만, 대회 참가자인 당신은 이 문제를 풀어야 한다.


$N$과 $M$이 주어졌을 때, 다음 조건을 모두 충족하는 길이 $N$의 수열 $A$를 구하라. 해당 수열은 항상 존재한다.




$0 \leq x \leq M$인 모든 정수 $x$에 대해, $\sum^r_{k=l} A_k = x$인 정수 $l$, $r$이 존재한다. $(1 \leq l \leq r \leq N)$


$x > M$인 모든 정수 $x$에 대해, $\sum^r_{k=l} A_k = x$인 정수 $l$, $r$이 존재하지 않는다. $(1 \leq l \leq r \leq N)$


$1 \leq i \leq N$인 모든 정수 $i$에 대해, $0 \leq A_i \leq 10^9$이다.


수열 $A$의 모든 원소는 정수이다.

## 예제 입력 1
```
4 3
```

## 예제 출력 1
```
0 0 1 2
```

## 예제 입력 2
```
5 5
```

## 예제 출력 2
```
0 2 0 2 1
```

## 예제 입력 3
```
4 4
```

## 예제 출력 3
```
0 2 1 1
```
