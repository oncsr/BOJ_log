# [1849. 순열](https://www.acmicpc.net/problem/1849)

## 문제
1부터 N까지의 수들이 한 번씩 쓰인 수열이 있다. 그 수열에서 i 앞에 있는 수 들 중, i보다 큰 수들의 개수를 A[i]라고 정의하자. A[i]가 주어져 있을 때, 원래 수열을 구하는 프로그램을 작성하여라.

## 예제 입력 1
```
8
5
0
1
2
1
2
0
0
```

## 예제 출력 1
```
2
7
3
5
4
1
8
6
```
