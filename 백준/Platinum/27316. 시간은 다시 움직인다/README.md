# [27316. 시간은 다시 움직인다](https://www.acmicpc.net/problem/27316)

## 문제
한별이는 어느 날 골동품 화살에 찔린 이후 초능력자가 되어 시간을 조종하는 신비한 힘, “백금 한별”을 얻게 되었다. 능력을 사용하는 동안에는 한별이가 시간을 다르게 느끼기 때문에, 마치 시간이 멈춘 것 같은 느낌을 받게 된다.


PS 애호가이기도 했던 한별이는, 이를 PS 문제를 푸는 것에 이용하고자 한다.


한별이는 PS 문제를 풀면서 서로 다른 시각에 찾아오는 여러 “고비”를 넘겨야 한다. 한별이는 이러한 고비를 능력을 사용하여 수월하게 넘기기로 했다.


한별이가 $i$ 번째로 능력을 사용하면 $i$초 동안 능력이 지속된다. 그러면 능력을 사용하는 순간부터 능력이 끝나는 순간 사이에 찾아오는 모든 고비를 수월하게 넘길 수 있게 된다. 한편, $i$ 번째로 사용한 능력이 끝나는 순간부터 $i$초가 지나야 지나야 다시 능력을 사용할 수 있게 된다. 능력을 사용하는 순간과 동시에 찾아오는 고비는 수월하게 넘길 수 있지만, 능력이 끝나는 순간에 만나는 고비는 수월하게 넘길 수 없다.


현재 시각은 $0$초이다. 각각의 고비가 찾아오는 시각이 주어질 때, 한별이가 최적으로 행동하면 모든 고비를 수월하게 넘길 수 있을지 판별하라.

## 예제 입력 1
```
6
1 3 4 7 8 9
```

## 예제 출력 1
```
YES
```

## 예제 입력 2
```
2
0 1
```

## 예제 출력 2
```
NO
```

## 예제 입력 3
```
2
3 4
```

## 예제 출력 3
```
YES
```
