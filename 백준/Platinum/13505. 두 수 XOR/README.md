# [13505. 두 수 XOR](https://www.acmicpc.net/problem/13505)

## 문제
N개의 수가 주어졌을 때, XOR한 값이 가장 큰 두 수를 찾는 프로그램을 작성하시오.


즉, A
1
, A
2
, ..., A
N
 중에서 i ≠ j이면서 A
i
 XOR A
j
 가 가장 큰 것을 찾아야 한다.

## 예제 입력 1
```
5
1 2 3 4 5
```

## 예제 출력 1
```
7
```

## 예제 입력 2
```
5
0 1 0 1 0
```

## 예제 출력 2
```
1
```

## 예제 입력 3
```
6
1 2 4 8 16 32
```

## 예제 출력 3
```
48
```
