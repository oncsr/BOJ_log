# [2261. 가장 가까운 두 점](https://www.acmicpc.net/problem/2261)

## 문제
2차원 평면상에 n개의 점이 주어졌을 때, 이 점들 중 가장 가까운 두 점을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
4
0 0
10 10
0 10
10 0
```

## 예제 출력 1
```
100
```
