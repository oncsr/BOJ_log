# [16903. 수열과 쿼리 20](https://www.acmicpc.net/problem/16903)

## 문제
0이 하나 포함되어 있는 배열 A가 있다. 이때, 다음 쿼리를 수행해야 한다.




1 x
: A에 x를 추가한다.


2 x
: A에서 x를 제거한다. A에 x가 두 개 이상 있는 경우에는 하나만 삭제한다. 항상 A에 x가 있는 쿼리만 주어진다.


3 x
: A에 포함된 각각의 원소와 x를 XOR 연산을 한 다음, 가장 큰 값을 출력한다.

## 예제 입력 1
```
10
1 8
1 9
1 11
1 6
1 1
3 3
2 8
3 3
3 8
3 11
```

## 예제 출력 1
```
11
10
14
13
```
