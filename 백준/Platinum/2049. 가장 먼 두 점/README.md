# [2049. 가장 먼 두 점](https://www.acmicpc.net/problem/2049)

## 문제
2차원 평면상에 n개의 점이 주어졌을 때, 이 점들 중 가장 먼 두 점을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
4
1 0
1 1
0 0
0 1
```

## 예제 출력 1
```
2
```

## 예제 입력 2
```
4
0 0
10 10
0 10
10 0
```

## 예제 출력 2
```
200
```
