# [1215. 잘못 작성한 요세푸스 코드](https://www.acmicpc.net/problem/1215)

## 문제
요세푸스 문제는 아래와 같이 풀 수 있다.



r := 0;
for i from 1 to n do
    r := (r + k) mod i;
return r;


하지만 상근이는 코드를 잘못 읽고 아래와 같이 작성했다.



r := 0;
for i from 1 to n do
    r := r + (k mod i);
return r;


n과 k가 주어졌을 때, 상근이의 잘못 작성한 소스를 실행시킨 결과를 출력하는 프로그램을 작성하시오.

## 예제 입력 1
```
5 3
```

## 예제 출력 1
```
7
```
