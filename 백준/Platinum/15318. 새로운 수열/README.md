# [15318. 새로운 수열](https://www.acmicpc.net/problem/15318)

## 문제
다음과 같은 수열 A가 주어졌을 때,


A = {a
0
, a
1
, ..., a
N-2
, a
N-1
}


새로운 수열 B는 다음과 같이 정의된다.


B = {b
0
, b
1
, ..., b
N-2
, b
N-1
}


\[b_i = \sum_{k=0}^{N-1}{((-1)^k \times (k+1) \times a_{(i+k)\%N})}\]


수열 A가 주어지면, 수열 B를 구해보자.

## 예제 입력 1
```
3
4 1 -3
```

## 예제 출력 1
```
-7 19 -8
```

## 예제 입력 2
```
5
-1 2 -3 4 -5
```

## 예제 출력 2
```
-55 35 -12 -12 35
```
