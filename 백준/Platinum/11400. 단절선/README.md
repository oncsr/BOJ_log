# [11400. 단절선](https://www.acmicpc.net/problem/11400)

## 문제
그래프가 주어졌을 때, 단절선을 모두 구해 출력하는 프로그램을 작성하시오.


단절선이란 그 간선을 제거했을 때, 그래프가 두 개 또는 그 이상으로 나누어지는 간선을 말한다. 즉, 제거했을 때 그래프의 connected component의 개수가 증가하는 간선을 말한다.

## 예제 입력 1
```
7 8
1 4
4 5
5 1
1 6
6 7
2 7
7 3
2 3
```

## 예제 출력 1
```
2
1 6
6 7
```
