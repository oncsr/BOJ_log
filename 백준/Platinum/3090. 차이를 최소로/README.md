# [3090. 차이를 최소로](https://www.acmicpc.net/problem/3090)

## 문제
정수 N개로 이루어진 배열 A가 주어진다. 상근이는 수열의 수 하나를 골라서 값을 1 감소시킬 수 있다. 단, 수는 1보다 작아질 수 없다.


상근이는 위의 감소시키는 연산을 최대 T번 하려고 한다. 이때, 인접한 수의 차이의 최댓값을 최소로 하는 프로그램을 작성하시오.

## 예제 입력 1
```
5 2
3 2 3 2 2
```

## 예제 출력 1
```
2 2 2 2 2
```

## 예제 입력 2
```
5 5
4 2 3 7 6
```

## 예제 출력 2
```
3 2 3 4 5
```

## 예제 입력 3
```
3 6
10 10 1
```

## 예제 출력 3
```
9 5 1
```
