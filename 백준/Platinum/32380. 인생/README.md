# [32380. 인생](https://www.acmicpc.net/problem/32380)

## 문제
$N$개의 정수로 이루어진 두 배열 $A$, $B$와 두 정수 $U$, $D$가 주어진다. 배열 $A$, $B$의 $i$번째 원소는 각각 $A_i$, $B_i$이다.


$f(n)$을 다음과 같이 정의한다.




아래 과정을 $i = 1, 2, \cdots, n$에 대해 순서대로 수행한다.
	


$A_i$와 $B_i$ 중 하나를 고른다.


이후 $i < j \le n$을 만족하는 모든 $j$에 대해, $A_i$를 골랐다면 $A_j$와 $B_j$의 값이 $U$만큼 증가하고, $B_i$를 골랐다면 $A_j$와 $B_j$의 값이 $D$만큼 감소한다.






$f(n)$은 수를 고르는 $2^n$가지 방법 중, 고른 수들의 합의 최솟값이다.




$1$ 이상 $N$ 이하의 모든 정수 $n$에 대해, $f(n)$의 값을 구해보자.

## 예제 입력 1
```
5 2 3
4 8 2 8 1
11 5 14 8 19
```

## 예제 출력 1
```
4
11
9
13
7
```
