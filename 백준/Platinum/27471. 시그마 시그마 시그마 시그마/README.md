# [27471. 시그마 시그마 시그마 시그마](https://www.acmicpc.net/problem/27471)

## 문제
이제 막 시그마를 배운 에릭은 시그마 여러 개를 이어붙이는 놀이에 빠져버렸다. 오늘도 놀이를 하던 에릭은 $2$와 $3$은 소수라서 불편하기 때문에 시그마를 $4$개 이어붙였다.


$N$개의 정수 $A_1, A_2, \cdots, A_N$이 주어질 때, 다음 값을 구하는 프로그램을 작성하시오.


$$\sum_{i=1}^{N-1} \sum_{j=i+1}^{N} \sum_{k=i}^{j-1} \sum_{l=k+1}^{j} \max(A_k, A_l)$$

## 예제 입력 1
```
3
3 1 4
```

## 예제 출력 1
```
18
```

## 예제 입력 2
```
7
3 1 4 1 5 9 2
```

## 예제 출력 2
```
672
```
