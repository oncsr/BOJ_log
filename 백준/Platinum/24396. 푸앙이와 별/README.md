# [24396. 푸앙이와 별](https://www.acmicpc.net/problem/24396)

## 문제
중앙대학교의 자랑스러운 청룡 푸앙이는 자신만의 별을 가지고 있다. 
N
개의 꼭짓점과 $\frac{N(N - 1)}{2}$개의 간선으로 이루어진 별은 무방향 완전 그래프이기 때문에 아름다운 모양을 띤다.


푸앙이는 모든 꼭짓점 간의 거리가 1이라는 사실에 매료되었다. 그러나 푸앙이를 싫어하는 명진이는 푸앙이의 별의 간선들 중 몇 개를 잘라버렸다. 푸앙이는 좌절했지만, 새로운 별에도 매력을 느끼기 위해 꼭짓점 간의 거리를 계산하기로 했다. 푸앙이를 도와 꼭짓점 간의 거리를 계산해주자!

## 예제 입력 1
```
5 4
1 4
1 3
3 5
1 2
```

## 예제 출력 1
```
0
2
3
2
1
```
