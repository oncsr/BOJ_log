# [30921. 🧩 N-Queen (Area)](https://www.acmicpc.net/problem/30921)

## 문제
N-Queen 문제는 크기가 
N
 × 
N
인 체스판 위에 
N
개의 퀸을 서로 공격할 수 없게 놓는 문제이다. 퀸은 같은 행, 열, 또는 대각선 위에 있는 다른 기물을 공격할 수 있다.


이 문제에서는 체스판이 찌그러져 있다. 즉 체스판은 그림과 같은 볼록사각형 ABCD로, 사각형의 네 변을 
N
등분하고, 네 선분 AB와 CD, AD와 BC에 있는 
N
 − 1개의 등분점들을 마주 보는 것끼리 이어서 만들어진 
N
2
개의 사각형 칸으로 구성된다.




이 체스판 위에 퀸 
N
개를 서로 공격하지 못하도록 놓는 방법들 가운데, 퀸이 위치한 칸의 넓이의 합이 최대가 되는 경우를 구하자.

## 예제 입력 1
```
4
6 2 8 2 7 0 6 0
```

## 예제 출력 1
```
0.75
```
