# [19568. 직사각형](https://www.acmicpc.net/problem/19568)

## 문제
당신은 $30 \times 30$ 크기의 표에 0 이상 $10^6$ 이하의 정수를 채워야 한다.


다음을 만족하는 정수 $i$를 찾을 수 있다고 하자. 표에 존재하는 $(\frac{30 \times 31}{2})^2$개의 직사각형 중, 그 안에 쓰인 수들의 합이 정확히 $i$인 것이 존재한다.
