# [16496. 큰 수 만들기](https://www.acmicpc.net/problem/16496)

## 문제
음이 아닌 정수가 N개 들어있는 리스트가 주어졌을 때, 리스트에 포함된 수를 나열하여 만들 수 있는 가장 큰 수를 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
5
3 30 34 5 9
```

## 예제 출력 1
```
9534330
```

## 예제 입력 2
```
5
0 0 0 0 1
```

## 예제 출력 2
```
10000
```
