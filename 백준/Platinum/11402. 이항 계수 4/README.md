# [11402. 이항 계수 4](https://www.acmicpc.net/problem/11402)

## 문제
자연수 \(N\)과 정수 \(K\)가 주어졌을 때 이항 계수 \(\binom{N}{K}\)를 \(M\)으로 나눈 나머지를 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
5 2 3
```

## 예제 출력 1
```
1
```

## 예제 입력 2
```
30 10 3
```

## 예제 출력 2
```
0
```

## 예제 입력 3
```
30 3 3
```

## 예제 출력 3
```
1
```

## 예제 입력 4
```
100 45 7
```

## 예제 출력 4
```
0
```

## 예제 입력 5
```
100 45 13
```

## 예제 출력 5
```
2
```
