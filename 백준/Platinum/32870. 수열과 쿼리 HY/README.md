# [32870. 수열과 쿼리 HY](https://www.acmicpc.net/problem/32870)

## 문제
길이가 $N$인 수열 $A_1$, $A_2$, $\cdots$, $A_N$이 주어진다. 이때, 다음 쿼리를 수행하는 프로그램을 작성하시오.




$m$: $A_1\bmod m$, $A_2\bmod m$, $\cdots$, $A_N\bmod m$ 중 최솟값과 최댓값을 출력하라. $(1\le m\le 300\, 000)$

## 예제 입력 1
```
10 6
9 55 2 4 26 37 81 16 34 46
1
5
6
7
9
12
```

## 예제 출력 1
```
0 0
0 4
1 4
2 6
0 8
1 10
```
