# [23034. 조별과제 멈춰!](https://www.acmicpc.net/problem/23034)

## 문제
교수님이 시험 기간에 조별 과제를 준비하셨다...! 가톨릭대학교의 조교 아리는 
N
명의 학생을 2개의 조로 구성하여 과제 공지를 하려 한다. 이때, 구성된 각 조의 인원은 1명 이상이어야 한다. 각 학생은 1~
N
의 정수 중 고유한 번호를 학번으로 갖는다.


모든 것이 귀찮은 아리는 각 조의 팀장에게만 공지를 전달한다. 아리는 
N
명의 학생 사이에 있는 총 
M
개의 회선의 리스트를 가지고 있다. 리스트에는 각 회선에 연결된 두 학생의 학번 
A
와 
B
, 연락에 필요한 비용 
C
가 적혀있다. 회선이 연결된 두 학생은 서로 연락이 가능하다. 아리가 각 팀장에게 공지를 전달하면, 각 팀장과 공지를 알게 된 팀원은 같은 조의 모든 팀원에게 공지 내용을 회선을 통해서만 전달한다. 어떤 학생이 팀장이 되어도 모든 학생은 공지 내용을 전달받을 수 있다.


아리는 공지 채팅방을 만들 생각은 안 하고, 모든 학생이 공지 내용을 알게 될 때까지 학생들이 연락하며 소요되는 비용의 총합 
T
의 최솟값을 알고 싶어졌다. 그것을 구하여 팀장을 정한 뒤 조를 구성하려고 한다.


아리는 다음과 같은
 Q
개의 질문을 한다.




X Y : 
X
와 
Y
가 팀장일 때, 
T
의 최솟값은 무엇인가?




Q
개의 질문에 답할 수 있는 프로그램을 작성하시오.
