# [11439. 이항 계수 5](https://www.acmicpc.net/problem/11439)

## 문제
자연수 \(N\)과 정수 \(K\)가 주어졌을 때 이항 계수 \(\binom{N}{K}\)를 \(M\)으로 나눈 나머지를 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
5 2 3
```

## 예제 출력 1
```
1
```

## 예제 입력 2
```
30 3 3
```

## 예제 출력 2
```
1
```

## 예제 입력 3
```
100 45 13
```

## 예제 출력 3
```
2
```

## 예제 입력 4
```
100 46 72
```

## 예제 출력 4
```
48
```

## 예제 입력 5
```
50 23 9
```

## 예제 출력 5
```
1
```
