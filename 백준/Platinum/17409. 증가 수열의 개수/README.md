# [17409. 증가 수열의 개수](https://www.acmicpc.net/problem/17409)

## 문제
크기가 N인 수열 A와 정수 K가 주어졌을 때, A의 증가하는 부분 수열 중에서 길이가 K인 것의 개수를 구해보자.

## 예제 입력 1
```
5 1
1 2 3 5 4
```

## 예제 출력 1
```
5
```

## 예제 입력 2
```
5 2
1 2 3 5 4
```

## 예제 출력 2
```
9
```

## 예제 입력 3
```
5 3
1 2 3 5 4
```

## 예제 출력 3
```
7
```

## 예제 입력 4
```
5 4
1 2 3 5 4
```

## 예제 출력 4
```
2
```

## 예제 입력 5
```
5 5
1 2 3 5 4
```

## 예제 출력 5
```
0
```
