# [24518. 잘 알려진 합 구하기](https://www.acmicpc.net/problem/24518)

## 문제
함수 $f(x)$를 $N$을 $x$로 나눴을 때의 몫으로, $g(x)$를 $x$를 $M$으로 나눴을 때의 나머지로 정의하자.


다음 값을 $1\,000\,000\,007$로 나눈 나머지를 구하라.


$$\sum_{i=1}^N f(i)×g(i)$$

## 예제 입력 1
```
4 3
```

## 예제 출력 1
```
9
```
