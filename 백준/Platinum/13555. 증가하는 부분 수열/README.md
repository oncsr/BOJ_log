# [13555. 증가하는 부분 수열](https://www.acmicpc.net/problem/13555)

## 문제
길이가 N인 수열 A
1
, A
2
, ..., A
N
와 정수 K 주어진다. 이때, 수열 A의 부분 수열 중에서 길이가 K이면서 증가하는 부분 수열의 개수를 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
4 3
1 2 2 10
```

## 예제 출력 1
```
2
```
