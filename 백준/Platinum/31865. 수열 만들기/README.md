# [31865. 수열 만들기](https://www.acmicpc.net/problem/31865)

## 문제
수열을 좋아하는 수민이는 $1$부터 $N$까지의 정수로 수열을 만들려고 한다.


수열은 다음과 같은 규칙으로 만들려고 한다.




원형으로 $N$개의 자리가 있고 $1$은 미리 하나의 자리에 채워 넣는다.


$2$부터 $N$까지의 수를 차례로 3번~4번 과정을 반복하며 채워 넣는다.


이미 놓인 수 중 하나를 골라 $p_i$로 정하고 $1$ 이상 $10^9$ 이하의 정수 중 하나를 골라 $x_i$로 정한다.


$p_i$가 써진 자리를 기준으로, 시계방향으로 돌았을 때 $x_i$번째 등장하는 빈자리를 찾아 그 자리에 수를 쓴다.


$1$이 쓰인 자리부터 시계방향으로, 차례로 자리에 적힌 $N$개의 수로 수열을 만든다.

## 예제 입력 1
```
8
1 3
2 5
2 2
3 2
1 3
6 2
1 1
```

## 예제 출력 1
```
1
3
8
2
5
4
7
6
```
