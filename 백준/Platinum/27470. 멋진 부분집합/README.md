# [27470. 멋진 부분집합](https://www.acmicpc.net/problem/27470)

## 문제
KSA 학생들은 
멋진 집합
을 좋아한다. 
멋진 집합
이란 모든 원소들의 최대공약수가 $1$보다 큰 정수들의 중복집합이다. 크기가 $N$인 중복집합이 주어졌을 때, 주어진 중복집합의 부분집합 중 크기가 $\left\lceil\cfrac{N}{2}\right\rceil$인 
멋진 집합
을 찾아보자!

## 예제 입력 1
```
4
2 3 4 5
```

## 예제 출력 1
```
YES
2 4
```

## 예제 입력 2
```
5
3 5 7 9 11
```

## 예제 출력 2
```
NO
```
