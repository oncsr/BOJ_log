# [18135. 겨울나기](https://www.acmicpc.net/problem/18135)

## 문제
가톨릭대학교 텔레토비 동산에는 겨울을 준비하는 다람쥐 다다가 살고 있다. 다다는 교내에서 자신만의 원형의 산책로를 가지고 있다.




다다는 원형의 산책로를
 N
 (1 ≤ 
N
 ≤ 2,000,000)개의 칸으로 구분하고 1번부터 
N
번 까지 번호를 매겼고, 한 개의 칸 혹은 연속된 여러 칸을
 M
 (1 ≤ 
M
 ≤ 1,000,000)개의 영역으로 지정했으며, 1번부터 
M
번까지 번호를 매겼다. 한 영역은 꽃밭, 건물 등 학교에서의 한 장소나 시설을 의미한다. 1번 칸은 무조건 1번 영역에 속하고 어떤 영역에도 속하지 않는 칸은 없다. 칸과 영역의 번호는 시계방향으로 순서대로 매겨지며, 다다의 이동 방향도 시계방향으로만 움직인다.


영역은 겹치지 않으며, 각 영역마다 도토리를 저장해 두었다.


겨울이 오기 전 산책로를 따라 걸으며 어떤 영역에 포함되는 칸을 지나면 그 영역에 저장된 도토리의 수량을 합하며 점검하고, 산책 중에 학생들에게 받은 도토리들을 저장하려고 한다. 겨울잠이 끝나고 다람쥐 다다가 눈을 뜰 수 있도록 다다를 도와주자.
