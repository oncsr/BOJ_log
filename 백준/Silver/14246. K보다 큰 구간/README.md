# [14246. K보다 큰 구간](https://www.acmicpc.net/problem/14246)

## 문제
$n$개의 자연수로 이루어진 수열이 주어질 때, 특정 구간 $[i,j]$ ($i≤j)$의 합이 $k$보다 큰 모든 쌍 $(i, j)$의 개수를 출력하시오.

## 예제 입력 1
```
5
1 2 3 2 1
7
```

## 예제 출력 1
```
3
```

## 예제 입력 2
```
5
1 1 1 1 1
2
```

## 예제 출력 2
```
6
```
