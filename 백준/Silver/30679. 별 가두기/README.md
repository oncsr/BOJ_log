# [30679. 별 가두기](https://www.acmicpc.net/problem/30679)

## 문제
스타는 별을 가두는 장치를 만들어 본인이 가지고 있는 별을 보관하려고 한다.


스타의 장치에는 $N \times M$ 크기의 격자가 있으며, 격자의 각 칸에는 양의 정수가 적혀 있다. 이 장치는 다음과 같은 과정을 거쳐 격자에 별을 가둔다.




첫 번째 열의 원하는 칸에 별을 올려둔다.


다음과 같은 과정을 따라 별이 이동한다.
	


별은 바라보고 있는 방향으로, 별이 놓인 칸에 적힌 수만큼 이동한다.


별이 바라보는 방향이 시계 방향으로 90도 돌아간다.








별은 처음에 오른쪽을 바라보고 있으며, 별이 격자 밖으로 나가지 않고 위 과정이 무한히 반복된다면 별을 성공적으로 가둔 것이다. 하지만 중간에 별이 격자 밖으로 나가게 된다면 별을 가두는 데에 실패한 것이다.




스타는 처음에 별을 어느 칸에 올려놔야 별을 가둘 수 있는지 궁금해졌다. 스타를 위해 별을 가둘 수 있는 칸을 알아 내보자.

## 예제 입력 1
```
4 4
1 2 2 3
2 2 1 1
3 2 1 2
1 2 2 2
```

## 예제 출력 1
```
1
2
```

## 예제 입력 2
```
6 5
3 1 1 2 1
4 2 1 3 1
4 1 1 3 3
4 3 3 1 2
5 1 1 2 2
3 2 3 3 3
```

## 예제 출력 2
```
3
2 3 4
```

## 예제 입력 3
```
5 6
5 2 3 3 1 1
4 2 2 1 1 3
3 1 2 2 1 3
5 3 2 1 1 2
3 2 3 3 3 2
```

## 예제 출력 3
```
0
```
