# [25516. 거리가 k이하인 트리 노드에서 사과 수확하기](https://www.acmicpc.net/problem/25516)

## 문제
n
개의 정점과 
n 
- 1개의 간선으로 구성된 트리 T가 있다. 정점 번호는 0부터 
n 
- 1까지이고 0번 정점이 루트이다. 모든 간선의 길이는 1이다. 트리 T의 각 정점에는 사과가 0개 또는 1개 놓여있다. 루트 노드에서 거리가 
k
이하인 노드에 있는 사과를 수확하려고 한다. 수확할 수 있는 사과 개수를 출력하자.

## 예제 입력 1
```
8 2
0 1
0 2
1 3
1 4
2 5
2 6
6 7
1 0 0 1 0 1 0 1
```

## 예제 출력 1
```
3
```
