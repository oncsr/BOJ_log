# [14425. 문자열 집합](https://www.acmicpc.net/problem/14425)

## 문제
총 N개의 문자열로 이루어진 집합 S가 주어진다.


입력으로 주어지는 M개의 문자열 중에서 집합 S에 포함되어 있는 것이 총 몇 개인지 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
5 11
baekjoononlinejudge
startlink
codeplus
sundaycoding
codingsh
baekjoon
codeplus
codeminus
startlink
starlink
sundaycoding
codingsh
codinghs
sondaycoding
startrink
icerink
```

## 예제 출력 1
```
4
```
