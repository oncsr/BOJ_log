# [27973. 지연 평가](https://www.acmicpc.net/problem/27973)

## 문제
지연이는 자신이 가진 배열 $S$의 상태를 평가하려고 한다. 
처음에 $S$에는 $1$부터 $1 \, 234 \, 567 \, 890 \, 123$까지의 모든 정수가 $1$개씩 들어 있다.


지연이를 도와 다음 명령을 수행하는 프로그램을 작성하시오.




0 x
: $S$의 모든 원소에 
x
를 더한다. $(-100 \, 000 \leq$ 
x
 $\leq 100 \, 000)$


1 x
: $S$의 모든 원소에 
x
를 곱한다. $(1 \leq$ 
x
 $\leq 100)$


2 n
: $S$에서 작은 원소부터 차례대로 
n
개를 제거한다. $(1 \leq$ 
n
 $\leq 100 \, 000 \, 000)$


3
: $S$에서 가장 작은 원소를 출력한다.




이 문제의 풀이를 작성하기 전에, 
예제
 밑의
