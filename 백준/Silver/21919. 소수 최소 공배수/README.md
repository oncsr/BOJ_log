# [21919. 소수 최소 공배수](https://www.acmicpc.net/problem/21919)

## 문제
행복이는 길이가 $N$인 수열 $A$에서 소수들을 골라 최소공배수를 구해보려고 한다.


행복이를 도와 이를 계산해주자.

## 예제 입력 1
```
5
2 3 5 6 8
```

## 예제 출력 1
```
30
```

## 예제 입력 2
```
4
4 16 64 256
```

## 예제 출력 2
```
-1
```
