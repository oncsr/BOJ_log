# [34671. 무토의 일본 여행](https://www.acmicpc.net/problem/34671)

## 문제
무토는 여행을 좋아하는 무사입니다. 무토는 관광지에 도착하면, 그곳에서 $1$시간 동안 머무릅니다. 한 곳에서만 있는 것이 심심했던 무토는 다음 질문을 $Q$번 하였습니다.




다음 조건을 만족하며 $s$번 관광지에서 $e$번 관광지로 이동하려고 해. 이때 최소 이동 시간은 몇 분일까?




$s$번 관광지에서 $e$번 관광지까지 이동할 때, 하나의 도로만 이용합니다.






무토의 질문을 해결해 주세요.

## 예제 입력 1
```
4 2 3
1 2 3
3 4 2
1 2
3 2
4 2
```

## 예제 출력 1
```
3
-1
-1
```

## 예제 입력 2
```
5 4 2
1 2 3
1 3 3
1 4 3
1 5 3
1 2
1 3
```

## 예제 출력 2
```
3
3
```
