# [30987. 하루 피부과](https://www.acmicpc.net/problem/30987)

## 문제
2023년, 하루는 피부과를 개원해 여드름 전용 레이저인 
HARU-LASER
를 이용해 사람들의 여드름을 치료해 주고 있다. 


여드름의 모양을 나타내는 이차함수는 $f(x) = ax^2+bx+c$로 정의되어 있고, 레이저에 설정되어 있는 일차함수를 $g(x) = dx+e$로 정의할 때, 레이저의 세기는 $\displaystyle{\int_{x_1}^{x_2} \{ f(x) - g(x) \}\mathrm{d}x}$로 정의된다.


이때, $x_1$, $x_2$는 여드름의 양 끝의 $x$좌표를 나타낸 것이며, $f(x)$와 $g(x)$의 정의역은 $(-\infty, \infty)$이다.


피부과를 개원한 지 얼마 안 된 하루는 레이저의 세기를 잘 계산하지 못한다. 여러분이 하루를 대신해 레이저의 세기를 계산해 주자.

## 예제 입력 1
```
0 1
3 4 2 8 3
```

## 예제 출력 1
```
-2
```

## 예제 입력 2
```
-3 3
-15 -4 5 -2 -5
```

## 예제 출력 2
```
-210
```
