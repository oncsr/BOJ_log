# [11727. 2×n 타일링 2](https://www.acmicpc.net/problem/11727)

## 문제
2×n 직사각형을 1×2, 2×1과 2×2 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.


아래 그림은 2×17 직사각형을 채운 한가지 예이다.

## 예제 입력 1
```
2
```

## 예제 출력 1
```
3
```

## 예제 입력 2
```
8
```

## 예제 출력 2
```
171
```

## 예제 입력 3
```
12
```

## 예제 출력 3
```
2731
```
