# [11053. 가장 긴 증가하는 부분 수열](https://www.acmicpc.net/problem/11053)

## 문제
수열 A가 주어졌을 때, 가장 긴 증가하는 부분 수열을 구하는 프로그램을 작성하시오.


예를 들어, 수열 A = {10, 20, 10, 30, 20, 50} 인 경우에 가장 긴 증가하는 부분 수열은 A = {
10
, 
20
, 10, 
30
, 20, 
50
} 이고, 길이는 4이다.

## 예제 입력 1
```
6
10 20 10 30 20 50
```

## 예제 출력 1
```
4
```
