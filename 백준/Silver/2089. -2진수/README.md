# [2089. -2진수](https://www.acmicpc.net/problem/2089)

## 문제
-2진법은 부호 없는 2진수로 표현이 된다. 2진법에서는 2
0
, 2
1
, 2
2
, 2
3
이 표현 되지만 -2진법에서는 (-2)
0
 = 1, (-2)
1

2
 = 4, (-2)
3

10진법의 수를 입력 받아서 -2진수를 출력하는 프로그램을 작성하시오.

## 예제 입력 1
```
-13
```

## 예제 출력 1
```
110111
```
