# [29941. Equality](https://www.acmicpc.net/problem/29941)

## 문제
Given four integers, insert between them the signs 
+
, 
-
, 
*
, or 
=
 to obtain a valid equality.

## 예제 입력 1
```
9?7?9?9
```

## 예제 출력 1
```
EI SAA
```

## 예제 입력 2
```
11?22?33?44
```

## 예제 출력 2
```
11=22+33-44
```

## 예제 입력 3
```
0?0?0?0
```

## 예제 출력 3
```
0+0=0+0
```
