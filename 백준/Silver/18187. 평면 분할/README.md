# [18187. 평면 분할](https://www.acmicpc.net/problem/18187)

## 문제
무한한 크기의 이차원 평면에, 여러분은 최대 
N
개의 직선을 그릴 수 있다.


여러분은 기울기가 -1, 0, 1인 직선만 그릴 수 있다.


직선을 이용하여 평면을 최대 몇 개의 영역으로 분할할 수 있는지 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
1
```

## 예제 출력 1
```
2
```

## 예제 입력 2
```
2
```

## 예제 출력 2
```
4
```

## 예제 입력 3
```
3
```

## 예제 출력 3
```
7
```
