# [15702. 중간고사 채점](https://www.acmicpc.net/problem/15702)

## 문제
이번 중간고사에는 총 N문제가 나왔고, 응시한 사람의 수는 M명이다. 각 문제의 배점과 각 사람의 결과가 주어졌을 때, 가장 높은 점수를 획득한 사람을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
4 4
10 20 30 40
1 O X X X
2 X O X X
3 X X O X
4 X X X O
```

## 예제 출력 1
```
4 40
```

## 예제 입력 2
```
4 8
10 20 30 40
1 O X X X
2 X O X X
3 X X O X
4 X X X O
80 O O O O
70 O O O O
60 O O O O
50 O O O O
```

## 예제 출력 2
```
50 100
```
