# [11659. 구간 합 구하기 4](https://www.acmicpc.net/problem/11659)

## 문제

## 예제 입력 1
```
5 3
5 4 3 2 1
1 3
2 4
5 5
```

## 예제 출력 1
```
12
9
1
```
