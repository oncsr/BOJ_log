# [15651. N과 M (3)](https://www.acmicpc.net/problem/15651)

## 문제
자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.




1부터 N까지 자연수 중에서 M개를 고른 수열


같은 수를 여러 번 골라도 된다.

## 예제 입력 1
```
3 1
```

## 예제 출력 1
```
1
2
3
```

## 예제 입력 2
```
4 2
```

```
1 3
1 4

2 4

3 4
4 1
4 2
4 3
4 4
```

```
3 3
```
