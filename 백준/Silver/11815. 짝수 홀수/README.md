# [11815. 짝수? 홀수?](https://www.acmicpc.net/problem/11815)

## 문제
B를 A로 나누었을 때 나머지가 0 이라면 A는 B의 약수라고 할 수 있다. (A > 0, B > 0) 예를 들면 15 의 약수는 1, 3, 5, 15 이다.


주어진 수가 가지는 약수 개수가 홀수인지 짝수인지 판별해보자.

## 예제 입력 1
```
2
4 5
```

## 예제 출력 1
```
1 0
```
