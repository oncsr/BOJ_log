# [32752. 수열이에요?](https://www.acmicpc.net/problem/32752)

## 문제
길이가 $N$인 정수열 $A$가 주어진다. 다음 조건을 만족하면 $A$는 
단조증가
하는 수열이다.




$1 \le i < j \le N$을 만족하는 모든 $(i, j)$에 대해 $A_i \le A_j$




$A$의 $L$번째 원소부터 $R$번째 원소의 순서를 임의로 재배치했을 때 
단조증가
하는 수열로 만들 수 있는지 확인해 보자.

## 예제 입력 1
```
5 2 4
1 4 2 3 5
```

## 예제 출력 1
```
1
```

## 예제 입력 2
```
5 2 4
1 6 3 8 3
```

## 예제 출력 2
```
0
```
