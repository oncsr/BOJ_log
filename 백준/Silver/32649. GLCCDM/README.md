# [32649. GLCCDM](https://www.acmicpc.net/problem/32649)

## 문제
$\operatorname{gcd}(d_1, \cdots, d_K) = A, \operatorname{lcm}(d_1, \cdots, d_K) = B$를 만족하는 서로 다른 양의 정수 $K$개로 이루어진 수열 $d_1, \cdots, d_K$를 구해보자.

## 예제 입력 1
```
2 36 3
```

## 예제 출력 1
```
4 6 18
```

## 예제 입력 2
```
2 8 5
```

## 예제 출력 2
```
-1
```
