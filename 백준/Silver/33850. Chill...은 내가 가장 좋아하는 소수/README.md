# [33850. Chill...은 내가 가장 좋아하는 소수](https://www.acmicpc.net/problem/33850)

## 문제
$2 \times n$ 격자의 각 칸에 양의 정수가 하나씩 적혀 있다. 산지니는 이 격자 위에 $2 \times 1$ 타일 또는 $1 \times 2$ 타일로 격자를 
빈틈없이
 채워 점수를 얻는 게임을 하고자 한다. 이때 한 타일이 덮고 있는 수의 합이 소수라면 $a$점, 소수가 아니라면 $b$점을 얻게 된다. 산지니가 얻을 수 있는 최고 점수를 알려주자.

## 예제 입력 1
```
2 2 3
2 2
3 3
```

## 예제 출력 1
```
6
```

## 예제 입력 2
```
2 3 2
2 2
3 3
```

## 예제 출력 2
```
6
```
