# [17419. 비트가 넘쳐흘러](https://www.acmicpc.net/problem/17419)

## 문제
🎶 DJ욱제는 
비트
에 몸을 맡기는 중이다. 🎶


DJ욱제
는 비트에 심취한 나머지, 
비트를 비틀어 제껴버리는
 문제를 내 버렸다!


N
자리 이진수 
K
가 주어진다. 
K
가 0이 아닐 때까지 아래의 연산을 적용했을 때, 연산이 일어나는 횟수를 구하시오.




K
 = 
K
-(
K
&((~
K
)+1))




아래는 위의 연산에 사용된 연산자에 대한 설명이다.




'+'는 산술 더하기 연산이다. (5 + 2 = 7)


'-'는 산술 빼기 연산이다. (5 - 2 = 3)


'&'는 비트 AND 연산이다. (1101 & 0111 = 0101)


'~'는 비트 NOT 연산이다. 켜진 비트를 끄고, 꺼진 비트를 켜는 연산이다. (~1101 = 0010)

## 예제 입력 1
```
5
10110
```

## 예제 출력 1
```
3
```
