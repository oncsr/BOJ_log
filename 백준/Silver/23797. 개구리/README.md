# [23797. 개구리](https://www.acmicpc.net/problem/23797)

## 문제
개구리의 울음소리는 
K
와 
P
가 번갈아서 나온다. 예를 들어, 
KPKP...
 또는 
PKPKPKPK...
 는 개구리의 울음소리지만, 
KKPP
는 아니다. 여러 개구리가 같이 울고있다. 다만, 개구리들이 우는 타이밍은 제멋대로라서 울음소리의 간격이 일정하지 않을 수 있다. 
K
와 
P
로 이루어진 문자열 \(S\)가 주어질 때, \(S\)에는 최소 몇 마리의 개구리가 울고있을까?

## 예제 입력 1
```
KKPKPPKKKP
```

## 예제 출력 1
```
3
```
