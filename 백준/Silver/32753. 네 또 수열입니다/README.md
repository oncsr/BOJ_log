# [32753. 네 또 수열입니다](https://www.acmicpc.net/problem/32753)

## 문제
정수 $1, 2, \cdots, N$이 각각 $K$개 있을 때, $N \times K$개의 수를 이용하여 다음 조건을 만족하는 길이가 $N \times K$인 수열을 만들어 보자.




$1 \le i \lt N \times K$를 만족하는 모든 정수 $i$에 대해 $A_1 + A_2 + \cdots + A_i = i$

## 예제 입력 1
```
1 2
```

## 예제 출력 1
```
1 1
```

## 예제 입력 2
```
3 4
```

## 예제 출력 2
```
-1
```
