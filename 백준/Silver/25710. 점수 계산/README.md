# [25710. 점수 계산](https://www.acmicpc.net/problem/25710)

## 문제
길이가 
N
인 배열 
a
가 주어진다. 배열 
a
의 
i
번째 원소를 
a
i
라고 정의하자.


다음 과정을 통해 배열에서 점수를 획득할 수 있다.




배열의 두 원소 
a
i
, 
a
j
를 선택한다. (1 ≤ 
i
 < 
j
 ≤ 
N
)


선택된 두 원소의 곱 
a
i
 ⨉ 
a
j
의 각 자릿수를 더한 값만큼 점수를 획득한다.




배열 
a
에서 획득할 수 있는 점수의 최댓값을 구하시오. 단, 점수는 한 번만 획득 가능하다.

## 예제 입력 1
```
2
3 8
```

## 예제 출력 1
```
6
```

## 예제 입력 2
```
4
1 19 3 10
```

## 예제 출력 2
```
12
```
