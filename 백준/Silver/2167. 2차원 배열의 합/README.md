# [2167. 2차원 배열의 합](https://www.acmicpc.net/problem/2167)

## 문제
2차원 배열이 주어졌을 때 (i, j) 위치부터 (x, y) 위치까지에 저장되어 있는 수들의 합을 구하는 프로그램을 작성하시오. 배열의 (i, j) 위치는 i행 j열을 나타낸다.

## 예제 입력 1
```
2 3
1 2 4
8 16 32
3
1 1 2 3
1 2 1 2
1 3 2 3
```

## 예제 출력 1
```
63
2
36
```
