# [33560. 수상한 어릿광대](https://www.acmicpc.net/problem/33560)

## 문제
수상한 어릿광대는 육면체 주사위를 굴려 주사위 눈에 따라 다음 행동들을 실행하며 점수를 얻는 게임이다.


게임을 시작할 때 기준 주사위를 한 번 굴릴 때마다 $1$점씩 얻고, 매 턴 시간이 $4$씩 흐른다. 턴을 시작할 때 기준 시각이 게임을 시작할 때로부터 $240$을 초과한 경우 주사위를 굴리기 전에 게임을 종료한다. 매 턴 눈에 따른 주사위의 효과를 실행한 후 점수를 얻고 시간이 흐른다.


주사위의 눈에 따른 효과는 다음과 같다.




1 - 즉시 게임이 종료되고 다음번 주사위를 굴릴 때 새로운 게임이 시작된다.


2 - 주사위를 한 번 굴릴 때마다 얻는 점수가 $1$점보다 큰 경우 이 값이 절반으로 줄어든다. $1$점인 경우에는 매 턴 흐르는 시간이 $2$ 증가한다.


3 - 특별한 효과가 없이 턴이 진행된다.


4 - 해당 턴에 시간을 $56$만큼 추가로 흐르게 만든다.


5 - 매 턴 흐르는 시간이 $1$보다 큰 경우 해당 값이 $1$ 감소한다.


6 - 주사위를 한 번 굴릴 때마다 얻는 점수가 $32$보다 작은 경우 이 값이 $2$배가 된다.




게임 종료 시 점수가 $35$점 이상 $65$점 미만인 경우

주어진 형식대로 굴렸을 때 얻을 수 있는 각 보상의 개수를 구해보자.
