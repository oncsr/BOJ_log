# [31870. 버블버블](https://www.acmicpc.net/problem/31870)

## 문제
민구는 원소의 개수가 $N$개이고 값이 서로 다른 정수 배열 $A$를 오름차순으로 만들고 싶다.


배열의 $i$번째 원소와 $i+1$번째 원소끼리 서로 위치를 바꿀 수 있고, 정렬 과정 중 언제든지 
최대 딱 한 번
 배열 전체의 순서를 뒤집을 수 있다.


원소를 교환하는 것, 배열 전체를 뒤집는 것 모두 $1$번의 횟수로 계산한다.


주어진 배열 $A$를 오름차순으로 만드는데 필요한 최소한의 횟수를 구하여라.

## 예제 입력 1
```
5
2 5 4 1 3
```

## 예제 출력 1
```
5
```
