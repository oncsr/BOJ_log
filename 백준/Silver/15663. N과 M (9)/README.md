# [15663. N과 M (9)](https://www.acmicpc.net/problem/15663)

## 문제
N개의 자연수와 자연수 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.




N개의 자연수 중에서 M개를 고른 수열

## 예제 입력 1
```
3 1
4 4 2
```

## 예제 출력 1
```
2
4
```

## 예제 입력 2
```
4 2
9 7 9 1
```

## 예제 출력 2
```
1 7
1 9
7 1
7 9
9 1
9 7
9 9
```

## 예제 입력 3
```
4 4
1 1 1 1
```

## 예제 출력 3
```
1 1 1 1
```
