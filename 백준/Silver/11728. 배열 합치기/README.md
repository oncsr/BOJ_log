# [11728. 배열 합치기](https://www.acmicpc.net/problem/11728)

## 문제
정렬되어있는 두 배열 A와 B가 주어진다. 두 배열을 합친 다음 정렬해서 출력하는 프로그램을 작성하시오.

## 예제 입력 1
```
2 2
3 5
2 9
```

## 예제 출력 1
```
2 3 5 9
```

## 예제 입력 2
```
2 1
4 7
1
```

## 예제 출력 2
```
1 4 7
```

## 예제 입력 3
```
4 3
2 3 5 9
1 4 7
```

## 예제 출력 3
```
1 2 3 4 5 7 9
```
