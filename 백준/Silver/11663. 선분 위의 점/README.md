# [11663. 선분 위의 점](https://www.acmicpc.net/problem/11663)

## 문제
일차원 좌표상의 점 N개와 선분 M개가 주어진다. 이때, 각각의 선분 위에 입력으로 주어진 점이 몇 개 있는지 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
5 5
1 3 10 20 30
1 10
20 60
3 30
2 15
4 8
```

## 예제 출력 1
```
3
2
4
2
0
```
