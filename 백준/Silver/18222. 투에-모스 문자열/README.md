# [18222. 투에-모스 문자열](https://www.acmicpc.net/problem/18222)

## 문제
0과 1로 이루어진 길이가 무한한 문자열 
X
가 있다. 이 문자열은 다음과 같은 과정으로 만들어진다.




X
는 맨 처음에 "0"으로 시작한다. 


X
에서 0을 1로, 1을 0으로 뒤바꾼 문자열 
X'
을 만든다.


X
의 뒤에 
X'
를 붙인 문자열을 
X
로 다시 정의한다. 


2~3의 과정을 무한히 반복한다.




즉, 
X
는 처음에 "0"으로 시작하여 "01"이 되고, "0110"이 되고, "01101001"이 되고, ⋯ 의 과정을 거쳐 다음과 같이 나타내어진다.


    "011010011001011010010110011010011001011001101001⋯⋯"


자연수 
k
가 주어졌을 때 
X
의 
k
번째에는 무슨 문자가 오는지 구하여라.

## 예제 입력 1
```
1
```

## 예제 출력 1
```
0
```

## 예제 입력 2
```
2
```

## 예제 출력 2
```
1
```

## 예제 입력 3
```
10
```

## 예제 출력 3
```
0
```
