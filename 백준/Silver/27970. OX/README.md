# [27970. OX](https://www.acmicpc.net/problem/27970)

## 문제
O
와 
X
로 이루어진 문자열이 주어진다. 모든 문자를 
X
로 만들 때까지 다음 연산을 반복할 때, 시행하는 연산의 횟수를 구하시오.




문자열의 가장 왼쪽에 있는 
O
를 
X
로 바꾸고, 그보다 왼쪽에 있는 
X
를 전부 
O
로 바꾼다.

## 예제 입력 1
```
OXO
```

## 예제 출력 1
```
5
```

## 예제 입력 2
```
X
```

## 예제 출력 2
```
0
```

## 예제 입력 3
```
OXXOXOXOXOOXXXXOXOOOXOOXXOOOXOXOXOXO
```

## 예제 출력 3
```
876151662
```
