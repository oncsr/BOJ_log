# [28138. 재밌는 나머지 연산](https://www.acmicpc.net/problem/28138)

## 문제
정수 $N$을 $m$으로 나눈 나머지가 $R$이 되도록 하는 모든 양의 정수 $m$의 합을 구하라.

## 예제 입력 1
```
16 4
```

## 예제 출력 1
```
18
```

## 예제 입력 2
```
6 0
```

## 예제 출력 2
```
12
```

## 예제 입력 3
```
12 8
```

## 예제 출력 3
```
0
```
