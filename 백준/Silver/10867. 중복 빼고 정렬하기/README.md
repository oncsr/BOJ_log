# [10867. 중복 빼고 정렬하기](https://www.acmicpc.net/problem/10867)

## 문제
N개의 정수가 주어진다. 이때, N개의 정수를 오름차순으로 정렬하는 프로그램을 작성하시오. 같은 정수는 한 번만 출력한다.

## 예제 입력 1
```
10
1 4 2 3 1 4 2 3 1 2
```

## 예제 출력 1
```
1 2 3 4
```
