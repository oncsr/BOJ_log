# [9711. 피보나치](https://www.acmicpc.net/problem/9711)

## 문제
피보나치 수열은 아래와 같이 표현된다.


1, 1, 2, 3, 5, 8, 13, 21, 34, ...


각 숫자는 앞의 두 숫자의 합으로 나타내는 것을 알 수 있다.


P와 Q 그리고 n이 주어질 때, P번째 피보나치 숫자를 Q로 나눈 나머지를 구하여라.

## 예제 입력 1
```
10
5 10
6 25
10 21
32 43
100 100
50 50
25 25
45 67
109 32
128 128
```

## 예제 출력 1
```
Case #1: 5
Case #2: 8
Case #3: 13
Case #4: 15
Case #5: 75
Case #6: 25
Case #7: 0
Case #8: 19
Case #9: 9
Case #10: 69
```
