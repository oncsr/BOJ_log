# [25511. 값이 k인 트리 노드의 깊이](https://www.acmicpc.net/problem/25511)

## 문제
n
개의 정점과 
n 
- 1개의 간선으로 구성된 트리 T가 있다. 정점 번호는 0부터 
n 
- 1까지이고 0번 정점이 루트이다. 간선에는 가중치가 없다. 트리 T에 있는 각 정점에는 서로 다른 값이 부여된다. 트리 T에서 정점에 부여된 값이 
k
인 노드의 깊이(depth)를 출력하자. 루트 정점의 깊이는 0이고 자식 정점의 깊이는 부모 정점의 깊이보다 1만큼 더 큰 값을 갖는다.

## 예제 입력 1
```
8 5
0 1
0 2
1 3
1 4
2 5
2 6
6 7
0 1 2 3 4 5 6 7
```

## 예제 출력 1
```
2
```
