# [34752. ZOAC 8](https://www.acmicpc.net/problem/34752)

## 문제
2025년 11월, 제8회 ZOAC(Zero One Algorithm Contest)이 개최된다.


이 대회에는 참가 번호를 통해 참가자의 행운을 시험하고 수상하는 독특한 전통이 있다.


참가자는 각 문자가 $0$부터 $9$까지의 정수로 이루어진 길이가 $10$인 문자열을 참가 번호로 받으며, 참가 번호의 앞의 숫자 $5$개와 뒤의 숫자 $5$개로 나누어 두 개의 정수 $X$와 $Y$를 만든다. 두 정수 $X$와 $Y$ 사이의 범위 $(\min(X,Y) \le i \le \max(X,Y))$ 에 존재하는 소수 $i$의 개수를 행운 점수라고 한다.


재민이는 계산된 행운 점수를 바탕으로 다음과 같은 수상 규칙에 따라 수상자를 결정하려고 한다.




행운상: 가장 높은 행운 점수를 얻은 참가자 중, 이름이 사전 순으로 가장 앞선 사람이 수상한다.


불운상: 가장 낮은 행운 점수를 얻은 참가자 중, 이름이 사전 순으로 가장 앞선 사람이 수상한다.


만약 최고점과 최저점이 같아 행운상과 불운상 수상자가 동일할 경우, 해당 참가자가 두 상을 모두 받는다.




$N$명의 참가자 정보가 주어질 때, 행운상과 불운상 수상자의 이름을 출력하는 프로그램을 작성해 보자.

## 예제 입력 1
```
3
ALICE 2546032483
BOB 1456425414
CHARLES 8548892650
```

## 예제 출력 1
```
BOB
CHARLES
```

## 예제 입력 2
```
2
DAVID 1234512346
GRACE 1000710007
```

## 예제 출력 2
```
GRACE
DAVID
```

## 예제 입력 3
```
3
EVE 9876587654
FRANK 8765498765
CHARLIE 1000110001
```

## 예제 출력 3
```
EVE
CHARLIE
```

## 예제 입력 4
```
3
CHOI 1000110001
KIM 6553565535
LEE 6553765537
```

## 예제 출력 4
```
LEE
CHOI
```
