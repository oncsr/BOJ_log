# [31908. 커플링 매치](https://www.acmicpc.net/problem/31908)

## 문제
종현이는 학교에 커플이 얼마나 많은지 확인하고자 한다.


커플은 왼손 약지에 남들과 구별되는 반지를 커플링으로 착용하고 있다는 사실을 알게 된 종현이는, 모든 재학생의 왼손 약지를 확인하기로 했다. 특징이 같은 반지를 착용한 사람이 정확히 두 명이라면, 종현이는 두 명을 커플 의심군으로 판단한다.


주어진 입력에 대해 종현이가 커플 의심군으로 판단할 사람들을 구하시오.

## 예제 입력 1
```
6
jonghyeon -
minkyu twisted_silver_matte
hyegeun thin_plated_gold
somin text_engraved_light_blue
daeun twisted_silver_matte
jiyoung thin_plated_gold
```

## 예제 출력 1
```
2
minkyu daeun
hyegeun jiyoung
```

## 예제 입력 2
```
6
eardrum_buster motdoen_goyangi
ksap motdoen_goyangi
beenzino chanel
bongjunho chanel
poison motdoen_goyangi
bts chanel
```

## 예제 출력 2
```
0
```
