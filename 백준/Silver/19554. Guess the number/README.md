# [19554. Guess the number](https://www.acmicpc.net/problem/19554)

## 문제
Write a program that guesses an integer X in the range 1, . . . , N.

## 예제 입력 1
```
5

-1

1

0
```

## 예제 출력 1
```
? 1

? 3

? 2

= 2
```
