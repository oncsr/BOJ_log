# [10211. Maximum Subarray](https://www.acmicpc.net/problem/10211)

## 문제
크기 N인 정수형 배열 X가 있을 때, X의 부분 배열(X의 연속한 일부분) 중 각 원소의 합이 가장 큰 부분 배열을 찾는 Maximum subarray problem(최대 부분배열 문제)은 컴퓨터 과학에서 매우 잘 알려져 있다.


여러분은 N과 배열 X가 주어졌을 때, X의 maximum subarray의 합을 구하자. 즉, max
1 ≤ i ≤  j ≤ N 
(X[i]+...+X[j])를 구하자.

## 예제 입력 1
```
2
5
1 2 3 4 5
5
2 1 -2 3 -5
```

## 예제 출력 1
```
15
4
```
