# [11652. 카드](https://www.acmicpc.net/problem/11652)

## 문제
준규는 숫자 카드 N장을 가지고 있다. 숫자 카드에는 정수가 하나 적혀있는데, 적혀있는 수는 -2
62
보다 크거나 같고, 2
62
보다 작거나 같다.


준규가 가지고 있는 카드가 주어졌을 때, 가장 많이 가지고 있는 정수를 구하는 프로그램을 작성하시오. 만약, 가장 많이 가지고 있는 정수가 여러 가지라면, 작은 것을 출력한다.

## 예제 입력 1
```
5
1
2
1
2
1
```

## 예제 출력 1
```
1
```

## 예제 입력 2
```
6
1
2
1
2
1
2
```

## 예제 출력 2
```
1
```
