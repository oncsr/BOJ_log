# [26122. 가장 긴 막대 자석](https://www.acmicpc.net/problem/26122)

## 문제
막대 자석 문자열은 문자 
N
과 
S
로만 구성되면서 다음과 같은 조건을 만족하는 문자열이다: 막대 자석 문자열에 등장하는 
N
의 개수와 
S
의 개수는 동일하며, 문자열의 앞쪽 절반을 구성하는 문자는 모두 
N
이거나 모두 
S
이다. 예를 들어, 
NS
, 
NNSS
, 
SSSNNN
 등은 막대 자석 문자열이지만, 
SNS
, 
NNNSS
, 
NSNS
 등은 막대 자석 문자열이 아니다.


윤이는 문자 
N
과 
S
로 구성된 문자열을 가지고 있다. 윤이는 이 문자열의 부분 문자열 중에서 가장 긴 막대 자석 문자열을 찾고자 한다. 부분 문자열이란 문자열의 연속된 일부를 의미한다. 윤이가 주어진 문자열의 부분 문자열 중에서 찾을 수 있는 가장 긴 막대 자석 문자열의 길이는 얼마인지 구하시오.

## 예제 입력 1
```
8
NSSSNNSN
```

## 예제 출력 1
```
4
```

## 예제 입력 2
```
5
NNNNN
```

## 예제 출력 2
```
0
```
