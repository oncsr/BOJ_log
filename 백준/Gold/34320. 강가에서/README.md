# [34320. 강가에서](https://www.acmicpc.net/problem/34320)

## 문제
강가에 $N$개의 가게가 일렬로 늘어져 있다.


이 세계에서 하루는 $1$시부터 $K$시까지 총 $K$시간으로 이루어져 있다. $i$번째 가게는 $S_i$시 1분 전에 문을 열고, $E_i$시 1분 후에 문을 닫는다. 즉, $S_i \le t \le E_i$인 정수 $t$에 대해 $i$번째 가게는 $t$시 정각에 영업하고 있다.


당신은 연속한 몇 개의 가게가 모두 열려 있을 때에 이들을 한 묶음으로 세고 있다. 이를테면, 1, 2, 4, 6, 7, 8번째 가게가 열려 있고 나머지 가게들이 닫혀있다고 하자. 이들은 총 세 묶음으로 이루어져 있는데, 1, 2번째 가게들이 한 묶음, 4번째 가게가 한 묶음, 6, 7, 8번째 가게들이 한 묶음이다.


매 시 정각에, 강가에 연속으로 열려 있는 가게들의 묶음의 수를 계산하는 프로그램을 작성하여라.

## 예제 입력 1
```
8 6
2 3 5 1 1 1 2 1
4 6 6 3 1 4 5 4
```

## 예제 출력 1
```
2 3 3 2 2 1
```
