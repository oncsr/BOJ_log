# [32656. 트리의 루트를 찾아라](https://www.acmicpc.net/problem/32656)

## 문제
$N$개의 정점으로 이루어진 트리가 있다.


형진이는 이 트리의 루트를 잊어버렸다. 유일하게 기억하는 것은 $LCA(a, b) = x$라는 것뿐이다.


트리의 루트로 가능한 정점 후보의 개수를 구해보자.

## 예제 입력 1
```
7
1 2
2 3
2 4
4 5
5 6
6 7
3 4 4
```

## 예제 출력 1
```
4
```

## 예제 입력 2
```
7
1 2
2 3
2 4
4 5
5 6
6 7
3 4 2
```

## 예제 출력 2
```
2
```

## 예제 입력 3
```
7
1 2
2 3
2 4
4 5
5 6
6 7
3 3 3
```

## 예제 출력 3
```
7
```
