# [9061. 두 직사각형](https://www.acmicpc.net/problem/9061)

## 문제
평면에 N 개의 점이 주어져 있을 때, 이 N 개의 점을 모두 포함하는 가능한 작은 “ 두 개의 직사각형” 을 찾으려 한다. 두 직사각형은 축에 평행하며, 여러분은 두 사각형 중 넓이가 큰 사각형의 넓이를 최소화해야 한다. 




단, 위 그림에서 보는 바와 같이 두 직사각형은 위와 같이 서로 겹치지 않아야 하며 (변 혹은 꼭짓점이 서로 접하는 것은 가능하다.) 두 사각형이 같은 모양이거나 같은 넓이를 가질 필요는 없다. 문제는 두 직사각형 중 큰 것의 넓이를 최소화하여 그 넓이를 출력하는 것이다. (참고로 위 그림은 큰 사각형의 넓이를 더 줄일 수 있으므로 정답은 아니다. 아래의 Sample Input/Output 의 두 번째 테스트 케이스를 참조하시오.) 또한, 직사각형은 그 너비 혹은 높이가 0 인 경우도 가능하며, 그런 경우 직사각형의 넓이는 0 이다.
