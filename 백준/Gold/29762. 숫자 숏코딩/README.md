# [29762. 숫자 숏코딩](https://www.acmicpc.net/problem/29762)

## 문제
양의 정수 $N$이 주어진다. 0부터 9까지의 숫자와 곱하기 
*
, 거듭제곱 
^
의 연산기호를 이용하여 $N$을 표현하는 가장 짧은 길이의 식을 구하시오.


식은 후위 표기법으로 나타내고, 각 수 또는 연산기호는 공백으로 구분한다. 단, 공백은 식의 길이에 포함하지 않는다.

## 예제 입력 1
```
1
```

## 예제 출력 1
```
1
```

## 예제 입력 2
```
10000
```

## 예제 출력 2
```
10 4 ^
```
