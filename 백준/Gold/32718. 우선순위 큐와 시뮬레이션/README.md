# [32718. 우선순위 큐와 시뮬레이션](https://www.acmicpc.net/problem/32718)

## 문제
$N$개의 원소가 들어 있는 우선순위 큐가 있다. 해당 큐의 
top
은 큐에 존재하는 원소의 최댓값으로 정의한다.


우선순위 큐를 이용해 시뮬레이션하려고 할 때, $i$번째 쿼리는 아래의 순서로 이루어진다.




우선순위 큐의 모든 원소에 대해 $A_i$를 더한다.


모든 원소에 대해 정수 $K$로 나눈 나머지를 취한다. 


top을 출력한다.




$T$개의 쿼리에 대한 결과를 구해보자.

## 예제 입력 1
```
5 10 5
7 3 10 8 4
4 8 6 7 8
```

## 예제 출력 1
```
8 9 8 9 7
```
