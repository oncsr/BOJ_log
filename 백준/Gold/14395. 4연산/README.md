# [14395. 4연산](https://www.acmicpc.net/problem/14395)

## 문제
정수 s가 주어진다. 정수 s의 값을 t로 바꾸는 최소 연산 횟수를 구하는 프로그램을 작성하시오.


사용할 수 있는 연산은 아래와 같다.




s = s + s; (출력: +)


s = s - s; (출력: -)


s = s * s; (출력: *)


s = s / s; (출력: /) (s가 0이 아닐때만 사용 가능)

## 예제 입력 1
```
7 392
```

## 예제 출력 1
```
+*+
```

## 예제 입력 2
```
7 256
```

## 예제 출력 2
```
/+***
```

## 예제 입력 3
```
4 256
```

## 예제 출력 3
```
**
```

## 예제 입력 4
```
7 7
```

## 예제 출력 4
```
0
```

## 예제 입력 5
```
7 9
```

## 예제 출력 5
```
-1
```

## 예제 입력 6
```
10 1
```

## 예제 출력 6
```
/
```
