# [33903. 괄호 붙이기](https://www.acmicpc.net/problem/33903)

## 문제
올바른 괄호 문자열은 다음과 같이 정의한다.




빈 문자열은 올바른 괄호 문자열이다.


S
가 올바른 괄호 문자열일 때, 
(
S
)
도 올바른 괄호 문자열이다.


S
와 
T
가 올바른 괄호 문자열이라면, 
ST
도 올바른 괄호 문자열이다.




예를 들어, 
()()
, 
(())
, 
(((()))(()))
 등은 올바른 괄호 문자열이고, 
((()
, 
)(
, 
)))()()((
 등은 그렇지 않다.


(
, 
)
로 이루어진 문자열 $A$, $B$가 주어진다.


$A$를 길이 $1$ 이상의 연속된 두 문자열 $A_1$, $A_2$로 순서를 유지해서 나눈 후 $A_1$, $B$, $A_2$를 이어붙여 새롭게 $A_1+B+A_2$라는 문자열을 만들었을 때,


올바른 괄호 문자열이 되도록 $A_1$, $A_2$를 나눌 수 있다면 $1$, 아니면 $0$을 출력한다.

## 예제 입력 1
```
((
))
```

## 예제 출력 1
```
0
```

## 예제 입력 2
```
(()
)
```

## 예제 출력 2
```
1
```
