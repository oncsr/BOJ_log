# [17355. Messi An-Gimossi](https://www.acmicpc.net/problem/17355)

## 문제
메시
는 축구 선수이다. 메시는 기분이 안 좋다.


messi: 
Messi An-Gimossi


메시의 기분은 예측 가능하다. N일의 기분 예보가 있다. 
기분이 나쁠 확률
이 각 날마다 기약분수 A/B 꼴로 주어진다.


기분이 나쁜 것이 독립적인 사건이라 할 때, 메시의 기분이 
매일 좋을 확률(N일 내내 좋을 확률)
을 기약분수로 구해 보자.
