# [33849. 정말 간단한 문제](https://www.acmicpc.net/problem/33849)

## 문제
수열 $x_i$와 $y_i$가 주어진다. 아래 간단한 질문에 답을 해보자.




$x_i$, $y_i$의 $l$번째 원소부터 $r$번째 원소까지의 합의 비 $\frac{\sum_{i = l}^{r} y_i}{\sum_{i = l}^{r} x_i}$가 최대가 될 때의 값과 그때의 구간 길이의 최댓값을 구하라.

## 예제 입력 1
```
5
4 1 2 3 2
6 1 3 4 3
```

## 예제 출력 1
```
3 2
1
```
