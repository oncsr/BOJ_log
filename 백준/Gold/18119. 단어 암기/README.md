# [18119. 단어 암기](https://www.acmicpc.net/problem/18119)

## 문제
준석이는 영어 단어를 외우려고 한다. 사전에는 
N
가지 단어가 적혀 있다. 모든 단어는 소문자이다. 단어 안에 있는 모든 알파벳을 알 때, 그 단어를 완전히 안다고 한다.


다음과 같은 쿼리들이 주어진다.




1 x
 : 알파벳 
x
를 잊는다.


2 x
 : 알파벳 
x
를 기억해 낸다.




처음에 모든 알파벳을 기억하는 상태고, 모음은 완벽하게 외웠기 때문에 절대 잊지 않는다.


각 쿼리마다 완전히 알고 있는 단어의 개수를 출력하여라.

## 예제 입력 1
```
5 10
apple
actual
banana
brick
courts
1 l
1 b
1 c
1 n
2 l
2 b
1 s
2 c
2 s
2 n
```

## 예제 출력 1
```
3
1
0
0
1
1
1
3
4
5
```
