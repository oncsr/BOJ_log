# [27727. 버튼 정렬](https://www.acmicpc.net/problem/27727)

## 문제
길이가 $N$인 수열 $A$와 버튼이 있다.


버튼을 누를 때마다 $A$에서 가장 작은 값을 갖는 원소를 하나 선택하여 $1$을 더한다. 그러한 원소가 여러 개라면 그 중 가장 
앞에 있는
 원소를 선택한다.

## 예제 입력 1
```
4
1 2 3 4
3
```

## 예제 출력 1
```
2
```
