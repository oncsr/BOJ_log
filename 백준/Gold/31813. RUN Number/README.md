# [31813. RUN Number](https://www.acmicpc.net/problem/31813)

## 문제
양의 정수 $x$의 모든 자릿수가 같은 숫자로 이루어져 있다면, 그런 $x$를 
RUN
 수라고 한다. 예를 들어, $4$, $111$, $888\, 888$은 
RUN
 수이지만, $27$, $334$, $100\, 000$은 아니다.


$N$자리 수 $K$가 주어지면, $K$를 최대 $(N+1)$개의 
RUN
 수의 합으로 표현하라. 이는 항상 가능함을 증명할 수 있다.

## 예제 입력 1
```
2
4 2024
3 506
```

## 예제 출력 1
```
4
999 999 22 4
3
444 55 7
```

## 예제 입력 2
```
2
17 99999999999999999
9 333666999
```

## 예제 출력 2
```
1
99999999999999999
3
333333333 333333 333
```
