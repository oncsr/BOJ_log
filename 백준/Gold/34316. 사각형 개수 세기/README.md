# [34316. 사각형 개수 세기](https://www.acmicpc.net/problem/34316)

## 문제
$N \times M$ 격자의 $(i, j)$ 칸에는 $1$ 이상 $9$ 이하의 숫자 $A_{i, j}$가 쓰여있다. 다음을 만족하는 $(x_1, x_2, y_1, y_2)$ 순서쌍의 개수를 세자.




$1 \leq x_1 < x_2 \leq N$, $1 \leq y_1 < y_2 \leq M$


$A_{x_1, y_1} + A_{x_1, y_2} + A_{x_2, y_1} + A_{x_2, y_2} = 20$

## 예제 입력 1
```
4 5
9 9 3 2 7
5 4 4 1 2
7 1 7 6 7
7 4 7 3 2
```

## 예제 출력 1
```
4
```
