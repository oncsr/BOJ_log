# [30160. 제곱 가중치](https://www.acmicpc.net/problem/30160)

## 문제
$N$개의 정수 $a_{1}, a_{2}, \ldots, a_{N}$이 주어진다.


$1^{2} \cdot a_{1}, 2^{2} \cdot a_{1} + 1^{2} \cdot a_{2}, \ldots, N^{2} \cdot a_{1} + (N-1)^{2} \cdot a_{2} + \ldots + 1^{2} \cdot a_{N}$을 각각 출력한다.


바꿔 말하면, $k = 1, 2, \ldots, N$에 대해서 $\sum_{i=0}^{k-1} (k-i)^2 \cdot a_{i+1}$를 출력한다.

## 예제 입력 1
```
4
1 2 3 4
```

## 예제 출력 1
```
1 6 20 50
```
