# [30242. 🧩 N-Queen (Easy)](https://www.acmicpc.net/problem/30242)

## 문제
N-Queen 문제는 크기가 $N \times N$인 체스판 위에 퀸 $N$개를 서로 공격할 수 없게 놓는 문제이다.


$N$이 주어졌을 때, 퀸을 놓는 방법 한 가지를 출력하는 것은 
쉽다
.


이 문제에서는 몇 개의 퀸이 이미 놓여있을 때, 퀸을 놓는 방법 한 가지를 출력해 보자.

## 예제 입력 1
```
3
0 0 0
```

## 예제 출력 1
```
-1
```

## 예제 입력 2
```
4
0 0 0 0
```

## 예제 출력 2
```
2 4 1 3
```

## 예제 입력 3
```
4
3 0 0 0
```

## 예제 출력 3
```
3 1 4 2
```

## 예제 입력 4
```
4
0 0 0 1
```

## 예제 출력 4
```
-1
```

## 예제 입력 5
```
10
3 0 0 0 0 0 0 0 0 0
```

## 예제 출력 5
```
3 1 6 9 5 10 8 4 2 7
```

## 예제 입력 6
```
19
0 0 2 0 0 0 0 0 0 0 0 7 0 0 0 0 0 0 10
```

## 예제 출력 6
```
1 5 2 6 17 12 8 19 14 18 15 7 3 11 9 4 13 16 10
```
