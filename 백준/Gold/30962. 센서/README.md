# [30962. 센서](https://www.acmicpc.net/problem/30962)

## 문제
다음과 같은 형태의 쿼리 $Q$개가 주어질 때, 각 쿼리에 대해 조건을 만족하는 정수 좌표 점의 개수를 출력하라.




$x_1, y_1, x_2, y_2, w$: 센서가 존재하는 원점 $(0, 0)$을 기준으로, 점 $(x_1, y_1)$를 포함하는 각도부터 시계방향으로 점 $(x_2, y_2)$를 포함하는 각도까지를 센서에서 스캔했을 때, 해당 각도에 포함된 점들 중 센서와의 거리가 $\sqrt{w}$여야 한다. (단, 세 점 $(0, 0),(x_1, y_1),(x_2, y_2)$은 일직선상에 있지 않으며, 시계방향으로 배치되어있다.)




센서의 크기는 무시해도 될 정도로 작다. 즉, $0$으로 생각해도 무방하다. 모든 입력은 정수이다.

## 예제 입력 1
```
3
1 3 2 1 5
1 1 2 1 5
1 3 1 1 10
```

## 예제 출력 1
```
2
1
1
```
