# [4276. 0이 몇 개?](https://www.acmicpc.net/problem/4276)

## 문제
n부터 m까지 숫자를 하나씩 종이에 작성할 때, 숫자 0을 몇 번 쓰는지 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
10 11
100 200
0 500
1234567890 2345678901
0 4294967295
-1 -1
```

## 예제 출력 1
```
1
22
92
987654304
3825876150
```
