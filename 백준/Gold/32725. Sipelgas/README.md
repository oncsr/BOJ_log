# [32725. Sipelgas](https://www.acmicpc.net/problem/32725)

## 문제
Robotsipelgas liigub mööda kuubi servi. Sipelgas peatub alati kuubi tipus ja ootab käsku: käsu 
V
 peale liigub ta järgmisse tippu mööda endast vasakul olevat serva, käsu 
P
 peale mööda paremal olevat serva.


Kirjutada programm, mis saab sipelga poolt seni täidetud käskude jada ja leiab sipelga jaoks lühima võimaliku tee tagasi tippu, kust ta liikumist alustas.

## 예제 입력 1
```
3
VVV
```

## 예제 출력 1
```
1
V
```
