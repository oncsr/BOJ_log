# [26601. 가장 작은 수](https://www.acmicpc.net/problem/26601)

## 문제
약수를 $2^N$개 갖는 양의 정수 중 가장 작은 수를 구하자.

## 예제 입력 1
```
0
```

## 예제 출력 1
```
1
```

## 예제 입력 2
```
1
```

## 예제 출력 2
```
2
```

## 예제 입력 3
```
7
```

## 예제 출력 3
```
83160
```
