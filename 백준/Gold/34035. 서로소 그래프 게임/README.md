# [34035. 서로소 그래프 게임](https://www.acmicpc.net/problem/34035)

## 문제
승민이와 찬영이는 서로소 그래프 게임을 하려고 한다. 이 게임은 간선이 없는 정점 $N$개로 시작하며, 두 사람은 번갈아 가며 다음과 같은 규칙으로 게임을 진행한다.




자신의 차례에는, 연결 요소에 속한 정점의 수가 서로소인 서로 다른 두 정점을 선택해 간선을 추가해야 한다.


간선을 추가할 수 있는 정점 쌍이 없다면, 해당 차례의 플레이어는 패배한다.




승민이가 먼저 시작하여 두 사람 모두 최선의 전략으로 게임을 진행할 때, 누가 승리하는지 판단하자.

## 예제 입력 1
```
2
2
3
```

## 예제 출력 1
```
kang
chan
```
