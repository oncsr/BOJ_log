# [1947. 선물 전달](https://www.acmicpc.net/problem/1947)

## 문제
이번 ACM-ICPC 대회에 참가한 모든 사람들은 선물을 하나씩 준비했다.


대회가 끝나고 난 후에 각자 선물을 전달하려고 할 때, 선물을 나누는 경우의 수를 구하는 프로그램을 작성하시오.


모든 사람은 선물을 하나씩 받으며, 자기의 선물을 자기가 받는 경우는 없다.

## 예제 입력 1
```
5
```

## 예제 출력 1
```
44
```

## 예제 입력 2
```
4
```

## 예제 출력 2
```
9
```

## 예제 입력 3
```
3
```

## 예제 출력 3
```
2
```

## 예제 입력 4
```
21
```

## 예제 출력 4
```
50944540
```
