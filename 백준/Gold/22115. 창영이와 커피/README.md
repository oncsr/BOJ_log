# [22115. 창영이와 커피](https://www.acmicpc.net/problem/22115)

## 문제
창영이는 커피를 좋아한다. 회사에 도착한 창영이는 아침 커피를 즐기려고 한다.


회사에는 
N
개의 커피가 각각 
하나씩
 준비되어 있고, 각 커피에는 카페인 함유량 
C
i
가 있다.


창영이는 
N
개의 커피 중 몇 개를 골라 정확히 
K
만큼의 카페인을 섭취하려고 한다.


창영이가 정확히 
K
만큼의 카페인을 섭취하기 위해서는 최소 몇 개의 커피를 마셔야 할까?

```
4 5
1 1 3 2
```

```
2
```

```

-1
```
