# [12878. Blocks](https://www.acmicpc.net/problem/12878)

## 문제
N개의 블록이 일렬로 놓여져 있습니다. 민호는 각각의 블록들을 빨, 주, 노, 초 네가지 색중 한가지 색으로 칠하려 합니다.


민호는 문득 빨간색으로 칠해진 블록의 개수와 노란색으로 칠해진 블록의 개수가 2로 나누어 떨어지게 칠하는 방법의 수가 몇 가지 인지 궁금해 졌습니다.


민호를 도와주는 프로그램을 작성해 봅시다.

## 예제 입력 1
```
1
```

## 예제 출력 1
```
2
```

## 예제 입력 2
```
2
```

## 예제 출력 2
```
6
```
