# [15487. A[j]-A[i]+A[l]-A[k]](https://www.acmicpc.net/problem/15487)

## 문제
크기가 N인 배열 A가 주어졌을 때, i < j < k < l을 만족하는 (i, j, k, l) 중에서 A[j]-A[i]+A[l]-A[k]의 최댓값을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
5
1 2 3 4 5
```

## 예제 출력 1
```
3
```

## 예제 입력 2
```
5
1 1 1 1 1
```

## 예제 출력 2
```
0
```

## 예제 입력 3
```
6
1 2 1 2 1 2
```

## 예제 출력 3
```
2
```

## 예제 입력 4
```
8
9 1 8 2 7 3 6 4
```

## 예제 출력 4
```
12
```

## 예제 입력 5
```
4
9 8 3 2
```

## 예제 출력 5
```
-2
```
