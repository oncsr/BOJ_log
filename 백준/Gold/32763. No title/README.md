# [32763. No title](https://www.acmicpc.net/problem/32763)

## 문제
이 문제는 
인터랙티브 
문제이다.


모든 원소의 절댓값이 $1$ 이상 $10$ 이하의 정수인 길이 $N$의 수열 $A_1$, $A_2$, $\cdots$, $A_N$이 있다. 당신은 $A$의 원소들을 모르지만, 인터랙터에게 두 원소의 합 또는 곱의 부호를 질문할 수 있다.


$N$번 이하의 질문을 통해 모든 원소의 부호를 알아내 보자.

## 예제 입력 1
```
4

+

+

-
```

## 예제 출력 1
```
? 1 + 2

? 2 * 3

? 3 + 4

! + - - +
```
