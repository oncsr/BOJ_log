# [32028. 이진 검색 트리 복원하기](https://www.acmicpc.net/problem/32028)

## 문제
찬솔이는 
N
개의 노드에 정수가 적혀있는 이진 검색 트리를 만들었다. 이진 검색 트리는 모든 노드 
i
에 대해서, 
i
의 왼쪽 자손 노드에 적힌 수는 모두 
i
에 적힌 수보다 작고, 
i
의 오른쪽 자손 노드에 적힌 수는 모두 
i
에 적힌 수보다 큰 이진 트리이다.


찬솔이는 모든 
i
에 대해서 이진 검색 트리의 
i
번 노드의 깊이 
H
i
와 이진 검색 트리의 
i
번 노드에 적힌 정수 
A
i
를 기록해두었다. 이때, 
A
i
는 모두 다르고 루트 노드의 깊이는 1이다. (1 ≤ 
i
 ≤ 
N
)


찬솔이는 ICPC World Finals에 다녀오는 동안 이진 검색 트리를 잃어버렸다. 찬솔이를 위해 이전에 기록해둔 정보를 이용하여 이진 검색 트리를 복원해주자.

```
3
1 2
2 1
3 2
```

```
-1 -1
```

```
-1
```

```
-1
```
