<p>국렬이는 연세대학교 신입생 프로그래밍 경진대회를 개최한 기념으로 공학관 근처 운동장에서 화석 발굴 이벤트를 진행하려고 했다. 화석 발굴 이벤트는 공학관 근처 운동장에 묻힌 화석을 찾는 이벤트이다. 그러나 공학관 내의 실험으로 인해 운동장의 일부가 인체에 매우 위험한 물질로 오염되었다.</p>
