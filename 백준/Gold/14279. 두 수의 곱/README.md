# [14279. 두 수의 곱](https://www.acmicpc.net/problem/14279)

## 문제
a, b, c가 주어졌을 때, A * B = C 이면서 |A - a| + |B - b| + |C - c|의 최솟값을 구하는 프로그램을 작성하시오. 이때, A, B, C는 양의 정수이어야 한다.


|x|는 x의 절댓값이다.

## 예제 입력 1
```
19 28 522
```

## 예제 출력 1
```
2
```

## 예제 입력 2
```
10 30 500
```

## 예제 출력 2
```
11
```

## 예제 입력 3
```
111 111 12321
```

## 예제 출력 3
```
0
```

## 예제 입력 4
```
1000 100 10
```

## 예제 출력 4
```
1089
```
