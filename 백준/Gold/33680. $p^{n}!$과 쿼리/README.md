# [33680. $p^{n}!$과 쿼리](https://www.acmicpc.net/problem/33680)

## 문제
다음 쿼리를 수행하는 프로그램을 작성하자.




$p$ $n$: 소수 $p$와 정수 $n$에 대해 $p^{n}!=p^k \times N$ 의 형태로 나타낼 때, $\gcd(p,N)=1$을 만족하는 정수 $k$의 값을 출력한다.

## 예제 입력 1
```
6
2 1
3 3
5 4
11 11
101 34890
999999937 1234
```

## 예제 출력 1
```
1
13
156
531166865
852347349
924979894
```
