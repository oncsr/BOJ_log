# [28099. 이상한 배열](https://www.acmicpc.net/problem/28099)

## 문제
길이가 $N$인 배열 $A$가 주어진다. 배열 $A$가 아래 조건을 만족한다면 이 배열 $A$를 
이상한 배열
이라 한다.




$A_i=A_j$를 만족하는 정수 $1 \le i,j \le N$와 $i < k < j$를 만족하는 정수 $k$에 대해, 항상 $A_k \le A_i$을 만족한다.




배열 $A$가 주어질 때 $A$가 
이상한 배열
인지 확인하여라.

## 예제 입력 1
```
6
3
1 2 3
5
1 1 2 2 3
6
1 2 3 1 2 3
9
6 3 2 3 6 4 1 4 6
7
4 3 2 3 1 3 4
9
6 3 1 3 6 4 1 4 6
```

## 예제 출력 1
```
Yes
Yes
No
Yes
Yes
No
```
