# [33851. DAG LCA](https://www.acmicpc.net/problem/33851)

## 문제
무가중치 방향 비순환 그래프
 $G$가 주어진다. 다음 쿼리를 수행하는 프로그램을 작성하시오.




$u$ $v$: 두 정점 $u$와 $v$로 가는 경로가 모두 존재하는 모든 정점에 대해 $u$와 $v$로 가는 두 최단 경로의 길이 중 작지 않은 값의 최솟값을 출력한다. 두 정점 $u$와 $v$로 가는 경로가 존재하는 정점이 없다면 대신 
-1
을 출력한다.

## 예제 입력 1
```
5 5 2
1 2
2 5
1 3
3 4
4 5
1 5
2 3
```

## 예제 출력 1
```
2
1
```

## 예제 입력 2
```
5 4 1
1 3
2 3
3 4
4 5
1 2
```

## 예제 출력 2
```
-1
```
