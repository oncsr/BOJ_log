# [34006. 사계절을 되찾은 자](https://www.acmicpc.net/problem/34006)

## 문제
카링
은 2023년 2월 23일에 추가된 메이플스토리 최상위 보스 몬스터로, 
카링
과 대결하는 동안 계절의 균형을 유지해야 하는 독특한 게이지 시스템을 가지고 있다.




계절의 균형: 게이지 시스템


노란색 게이지를 
봄의 기운
, 파란색 게이지를 
여름의 기운
, 보라색 게이지를 
가을의 기운
이라고 한다. 각 게이지의 양은 $0$ 이상 $2N$ 이하의 정수로 표현할 수 있다. 각 게이지의 양을 합치면 항상 $3N$이며 게이지 총량은 항상 변하지 않는다고 가정한다. 처음에는 
봄의 기운
은 
$A$
, 
여름의 기운
은 
$B$
, 
가을의 기운
은 
$C$
만큼 있다.


카링
이 공격하는 종류에 따라 피격 당했을 때, 게이지 변화가 다르다. 게이지 변화는 다음 $3$가지 중 하나에 해당한다.




궁기의 공격
: 
봄의 기운
이 
$a$
만큼 상승한다. 동시에 
가을의 기운
이 
$a$
만큼 감소한다.


도올의 공격
: 
여름의 기운
이 
$b$
만큼 상승한다. 동시에 
봄의 기운
이 
$b$
만큼 감소한다.


혼돈의 공격
: 
가을의 기운
이 
$c$
만큼 상승한다. 동시에 
여름의 기운
이 
$c$
만큼 감소한다.




계절의 균형을 이루려면 한 게이지라도 $0$ 이하가 되거나, $2N$ 이상이 되지 않아야 한다.




$N=500$일 때, 
봄의 기운
이 
$1\,000$
, 
가을의 기운
이 
$0$
이 되어 균형이 깨진 상태


따라서 계절의 균형을 안전하게 유지하기 위해 각 게이지의 양을 $N$에 가깝게 유지하는 것이 중요하다. 계절의 균형을 맞추는 방법을 터득해 
카링
 게이지 시스템을 파훼 해보자! 처음 상태의 게이지가 주어졌을 때, 모두 $N$으로 맞추기 위한 최소 피격 횟수를 구하고, 그 공격을 어떤 순서로 피격 당해야 하는지 구해보자.

## 예제 입력 1
```
500
250 300 950
50 50 50
```

## 예제 출력 1
```
13
AAAAAAAAABBBB
```

## 예제 입력 2
```
500
507 163 830
163 830 100
```

## 예제 출력 2
```
-1
```
