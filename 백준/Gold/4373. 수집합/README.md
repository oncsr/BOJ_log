# [4373. 수집합](https://www.acmicpc.net/problem/4373)

## 문제
정수 집합 S가 주어졌을 때, a + b + c = d를 만족하는 가장 큰 d를 구하는 프로그램을 작성하시오. 이때, a, b, c, d는 S의 원소이며, 서로 다른 수이다.

## 예제 입력 1
```
5
2
3
5
7
12
5
2
16
64
256
1024
0
```

## 예제 출력 1
```
12
no solution
```
