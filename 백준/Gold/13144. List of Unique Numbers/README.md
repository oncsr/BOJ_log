# [13144. List of Unique Numbers](https://www.acmicpc.net/problem/13144)

## 문제
길이가 N인 수열이 주어질 때, 수열에서 연속한 1개 이상의 수를 뽑았을 때 같은 수가 여러 번 등장하지 않는 경우의 수를 구하는 프로그램을 작성하여라.

## 예제 입력 1
```
5
1 2 3 4 5
```

## 예제 출력 1
```
15
```

## 예제 입력 2
```
5
1 2 3 1 2
```

## 예제 출력 2
```
12
```

## 예제 입력 3
```
5
1 1 1 1 1
```

## 예제 출력 3
```
5
```
