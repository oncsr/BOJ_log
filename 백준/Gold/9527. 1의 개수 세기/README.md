# [9527. 1의 개수 세기](https://www.acmicpc.net/problem/9527)

## 문제
두 자연수 A, B가 주어졌을 때, A ≤ x ≤ B를 만족하는 모든 x에 대해 x를 이진수로 표현했을 때 1의 개수의 합을 구하는 프로그램을 작성하시오.


즉, f(x) = x를 이진수로 표현 했을 때 1의 개수라고 정의하고, 아래 식의 결과를 구하자.


\[\sum_{x=A}^{B}{f(x)}\]

## 예제 입력 1
```
2 12
```

## 예제 출력 1
```
21
```
