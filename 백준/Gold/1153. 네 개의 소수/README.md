# [1153. 네 개의 소수](https://www.acmicpc.net/problem/1153)

## 문제
임의의 자연수가 주어지면, 이를 네 개의 소수의 합으로 분해하는 프로그램을 작성하시오. 예를 들어 38 = 5 + 7 + 13 + 13이 된다.

## 예제 입력 1
```
38
```

## 예제 출력 1
```
5 7 13 13
```
