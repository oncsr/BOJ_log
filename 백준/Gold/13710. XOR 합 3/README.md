# [13710. XOR 합 3](https://www.acmicpc.net/problem/13710)

## 문제
수열의 XOR 합이란 수열에 들어있는 모든 원소를 다 XOR한 값이다.


수열 A 주어졌을 때, A의 모든 연속하는 부분 수열의 XOR 합을 더한 값을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
2
1 2
```

## 예제 출력 1
```
6
```
