# [27739. 수열 재배열](https://www.acmicpc.net/problem/27739)

## 문제
$1$부터 $N$까지의 수가 하나씩 임의로 배열된 길이 $N$의 수열 $A_1, A_2,..., A_N$이 주어진다. 주어진 수열에서 연속된 $K$개의 수를 고른 후, 선택한 $K$개의 수의 구간 내에서 $K$개의 수를 자유롭게 재배열하는 연산을 한 번 수행한다.


연산을 수행한 후, 수열에서 만들 수 있는 연속된 증가하는 부분 수열의 최대 길이를 구하시오.


증가하는 수열이란, 모든 $i < j$에 대해 $A_i < A_j$인 수열을 말한다.

## 예제 입력 1
```
5 3
2 1 5 3 4
```

## 예제 출력 1
```
4
```

## 예제 입력 2
```
7 3
3 2 1 7 4 5 6
```

## 예제 출력 2
```
5
```
