# [2171. 직사각형의 개수](https://www.acmicpc.net/problem/2171)

## 문제
2차원 평면 위에 N(1 ≤ N ≤ 5,000)개의 점들이 있다. 이런 점들 중 서로 다른 네 개의 점을 잡아서 연결하면 사각형이 만들어진다. 이러한 사각형들 중에서, x축과 y축에 평행한 직사각형의 개수를 구하는 프로그램을 작성하시오. 직사각형의 넓이는 양수이어야 한다.

## 예제 입력 1
```
6
0 0
0 1
0 2
1 0
1 1
1 2
```

## 예제 출력 1
```
3
```
