# [25647. Oscar's Round Must Have a Constructive Problem](https://www.acmicpc.net/problem/25647)

## 문제
Grammy has a sequence $A$ of length $n$.


Please find a permutation $P$ such that $P_i \neq A_i$ for all $i$.

## 예제 입력 1
```
3
3
3 3 3
3
3 2 1
6
1 1 4 5 1 4
```

## 예제 출력 1
```
NO
YES
1 3 2
YES
4 5 1 2 3 6
```
