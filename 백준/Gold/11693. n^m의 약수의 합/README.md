# [11693. n^m의 약수의 합](https://www.acmicpc.net/problem/11693)

## 문제
두 정수 n, m이 주어졌을 때, n
m
의 모든 약수의 합을 출력하는 프로그램을 작성하시오.

## 예제 입력 1
```
2 1
```

## 예제 출력 1

3
```

2 10
```

```
2047
```

3 2
```

```

5 10

12207031

12345 67890
```

336000773
```
