# [14136. Višekratnici](https://www.acmicpc.net/problem/14136)

## 문제
Zadan je niz prirodnih brojeva. Pronañite duljinu najduljeg rastućeg podniza u kojem je svaki broj višekratnik prethodnog broja.

## 예제 입력 1
```
5
1
2
3
4
5
```

## 예제 출력 1
```
3
```

## 예제 입력 2
```
5
3
9
6
3
15
```

## 예제 출력 2
```
2
```
