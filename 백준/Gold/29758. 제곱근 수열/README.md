# [29758. 제곱근 수열](https://www.acmicpc.net/problem/29758)

## 문제
양의 정수 $N$과 $L$에 대해, 다음 조건을 만족하는 수열 $A_1, A_2, \cdots, A_L$을 
제곱근 수열
이라고 한다.




$A_1 = N$, $A_L = 1$


모든 $1 \le i < L$에 대해, $\sqrt{A_i} > A_{i+1}$이다.


모든 항은 양의 정수이다.




$N$과 $L$이 주어질 때, 제곱근 수열의 개수를 구하시오.

## 예제 입력 1
```
2
3 2
10 3
```

## 예제 출력 1
```
1
2
```
