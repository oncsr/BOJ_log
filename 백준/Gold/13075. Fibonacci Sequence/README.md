# [13075. Fibonacci Sequence](https://www.acmicpc.net/problem/13075)

## 문제
Your job is to take an integer x as the input and compute f(x) mod 10
9
, where f(x) is the x-th value in the well-known Fibonacci sequence. 


Note that Fibonacci sequence is defined as follows:


f(1) = f(2) = 1

f(k) = f(k-1) + f(k-2) for any k > 2

```
3750000000
3750000001
281474976710656
```
