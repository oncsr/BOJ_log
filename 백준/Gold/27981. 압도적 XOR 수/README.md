# [27981. 압도적 XOR 수](https://www.acmicpc.net/problem/27981)

## 문제
양의 정수 $A$에 대해 $A$보다 큰 $2$의 제곱수 중 가장 작은 값을 $P$라고 할 때, $B = A \oplus (P-1)$라고 하자.


이 때, 양의 정수 $K$에 대해 $A \geq B \times (2^K-1)$일 경우 $A$를 
압도적 XOR 수
라고 한다.


$N$과 $K$가 주어지면 $1$부터 $N$까지의 정수 중 
압도적 XOR 수
의 개수를 구해보자.

## 예제 입력 1
```
243 4
```

## 예제 출력 1
```
22
```

## 예제 입력 2
```
243 7
```

## 예제 출력 2
```
7
```

## 예제 입력 3
```
1000000000000000 2
```

## 예제 출력 3
```
437050046578689
```
