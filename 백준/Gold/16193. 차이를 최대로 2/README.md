# [16193. 차이를 최대로 2](https://www.acmicpc.net/problem/16193)

## 문제
N개의 정수로 이루어진 배열 A가 주어진다. 이때, 배열에 들어있는 정수의 순서를 적절히 바꿔서 다음 식의 최댓값을 구하는 프로그램을 작성하시오.


|A[0] - A[1]| + |A[1] - A[2]| + ... + |A[N-2] - A[N-1]|

## 예제 입력 1
```
6
20 1 15 8 4 10
```

## 예제 출력 1
```
62
```
