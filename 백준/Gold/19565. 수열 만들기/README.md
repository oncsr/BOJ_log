# [19565. 수열 만들기](https://www.acmicpc.net/problem/19565)

## 문제
아래 성질을 만족하는 가장 긴 수열 $A$를 출력해 보자.




$A$의 첫 수와 마지막 수는 모두 1이고, 나머지 수는 1 이상 $N$ 이하인 정수이다.


$1 \le i < j < |A|$일 때, $A_i \neq A_j$, $A_{i+1} \neq A_{j+1}$ 둘 중 하나는 반드시 성립한다.

## 예제 입력 1
```
3
```

## 예제 출력 1
```
10
1 1 2 2 3 1 3 3 2 1
```
