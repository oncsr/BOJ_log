# [4320. 완전 P제곱수](https://www.acmicpc.net/problem/4320)

## 문제
x가 정수 b의 제곱(x = b
2
)으로 나타낼 수 있으면 x를 완전 제곱수라고 한다. 또, x를 정수 b의 세제곱(x = b
3
)으로 나타낼 수 있으면 x를 완전 세제곱수라고 한다. 즉, x를 b의 p제곱(x = b
p
)으로 나타낼 수 있으면, x를 완전 p제곱수라고 한다.



	x가 주어졌을 때, 가장 큰 p를 찾는 프로그램을 작성하시오.

## 예제 입력 1
```
17
1073741824
25
0
```

## 예제 출력 1
```
1
30
2
```
