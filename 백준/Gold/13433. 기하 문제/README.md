# [13433. 기하 문제](https://www.acmicpc.net/problem/13433)

## 문제
N개의 원을 일직선 위에 겹치지 않게 놓으려고 한다. 모든 원이 직선과 접해야 한다.


가장 왼쪽 원과 직선이 닿는 점과 가장 오른쪽 원과 직선이 닿는 점 사이의 거리의 최솟값을 구하는 프로그램을 작성하시오.

## 예제 입력 1
```
2
1 2
```

## 예제 출력 1
```
2.8284271247461903
```

## 예제 입력 2
```
3
7 7 7
```

## 예제 출력 2
```
28
```

## 예제 입력 3
```
3
10 20 30
```

## 예제 출력 3
```
62.92528739883945
```

## 예제 입력 4
```
5
100 100 11 11 25
```

## 예제 출력 4
```
200
```
