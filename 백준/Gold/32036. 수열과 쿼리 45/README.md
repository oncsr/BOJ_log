# [32036. 수열과 쿼리 45](https://www.acmicpc.net/problem/32036)

## 문제
길이가 $2 \times 10^9 + 1$ 인 배열 $A$ 가 있다. $A$ 의 인덱스는 $[-10^9, 10^9]$ 범위의 정수이다. 초기 $A$ 의 모든 원소는 $0$ 이다. (즉, $A[-10^9] = A[-10^9+1] = \ldots = A[10^9] = 0$)


이 배열에 $q$ 개의 쿼리가 주어진다. 쿼리는 다음과 같은 두 종류이다:




1 x y
: 모든 $-10^9 \le i \le 10^9$ 에 대해 $A[i] = A[i] + |i - x| + y$ 를 대입한다.


2
: $A$ 의 최솟값이 등장하는 가장 첫 지점 $m$와, $A[m]$ 를 출력한다. (첫 지점이라 함은 인덱스가 최소인 지점을 뜻한다.)

## 예제 입력 1
```
6
1 1 2
2
1 10 -4
2
1 5 0
2
```

## 예제 출력 1
```
1 2
1 7
5 7
```
