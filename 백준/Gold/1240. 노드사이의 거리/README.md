# [1240. 노드사이의 거리](https://www.acmicpc.net/problem/1240)

## 문제
$N$개의 노드로 이루어진 트리가 주어지고 M개의 두 노드 쌍을 입력받을 때 두 노드 사이의 거리를 출력하라.

## 예제 입력 1
```
4 2
2 1 2
4 3 2
1 4 3
1 2
3 2
```

## 예제 출력 1
```
2
7
```
