# [13010. h(n)](https://www.acmicpc.net/problem/13010)

## 문제
성관이는 양의 정수에서 동작하는 함수 h를 공부하고 있다.


먼저, d(n)을 n의 서로 다른 양의 약수의 개수로 정의한다.


h(n) = n
d(n)
이다. 즉, n을 d(n)만큼 제곱한 것이 h(n) 이다.


예를 들어, d(6) = 4이기 때문에, h(6) = 6
4
 = 1296이 된다.


정수 n이 주어졌을 때, h(x) = n을 만족하는 가장 작은 x를 찾는 프로그램을 작성하시오.
