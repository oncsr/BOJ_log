# [1355. 구멍난 케이크 자르기](https://www.acmicpc.net/problem/1355)

## 문제
태수는 생일 선물로 특별한 케이크를 받았다. 이 케이크를 위에서 보면 가운데 정사각형 구멍이 뚫려 있는 정사각형 모양이다.


케이크를 수평으로 H번, 수직으로 V번 자르려고 한다. 수평으로 자를 때는 x축과 평행하면서 점 (0, h
i
