# [32404. 일이 커졌어](https://www.acmicpc.net/problem/32404)

## 문제
정우는 정수 $1,2,\cdots ,N$을 원하는 순서로 나열해 순열 $p=[p_1,p_2,\cdots ,p_N]$을 만든 후, 이 순열을 이용해서 정우가 가지고 있는 정수 $X$를 키우려고 한다. $X$의 초깃값은 $1$이다.


정우는 $i$를 $1$부터 $N$까지 $1$씩 증가시키며 $p_i$에 따라 정수 $X$를 키울 수 있는데, 그 방법은 다음과 같다.




$i$가 홀수라면 $X$에 $p_i$를 곱한다.


$i$가 짝수라면 $X$에 $p_i$를 더한다.




예를 들어, $N=4$이고 순열 $p=[1,2,3,4]$일 때, $X$는 $1\times 1=1$, $1+2=3$, $3\times 3=9$, $9+4=13$으로 변한다.


정수 $N$이 주어질 때, 정수 $1,2,\cdots ,N$을 어떻게 나열해야 정수 $X$를 최대로 키울 수 있는지 구해보자.

## 예제 입력 1
```
2
```

## 예제 출력 1
```
2 1
```
