# [35498. 정렬과 쿼리](https://www.acmicpc.net/problem/35498)

## 문제
길이가 $N$인 수열 $A_1, A_2, \cdots, A_N$이 주어진다. 다음 쿼리를 수행하는 프로그램을 작성하자.




1 i j v
: $A_i, A_{i+1}, \cdots, A_j$의 구간에 $v$씩 더한다.


2
: 수열이 단조증가수열이면 
YES
를, 아니면 
NO
를 출력한다.




단조증가수열이란, $1 \le i \le N-1$를 만족하는 모든 $i$에 대해 $A_i \le A_{i+1}$인 수열을 의미한다.

## 예제 입력 1
```
5 5
3 2 3 4 5
2
1 2 5 2
2
1 4 5 -3
2
```

## 예제 출력 1
```
NO
YES
NO
```
